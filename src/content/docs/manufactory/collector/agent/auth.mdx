## Authentication

Before you start sending data to Tallyfy Manufactory[^1], you'll need to add the following values to the config file.

### Required keys

- `org_id` (required) string - Manufactory org ID.
- `project_id` (required) number - The project ID associated with this schema.
- `project_key`[^2] (required) string - Unique API key for this project.

### config.yaml

```yaml
org_id: xyz
project_id: 12345
project_key: b0iqA8ezfhPgacy32PybGWghRkGrOo9H
```

`manufactory-agent` won't find `config.yaml` unless both files are in the same folder.
