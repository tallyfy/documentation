## Agent route

The Tallyfy Manufactory collector lets you send actors data and events, which you can view in our custom interface.

The Manufactory agent helps when you need to off-load your API from sending events. It also works on any IoT device with limited computing power or connectivity.

Please discuss the agent with us if you're interested https://tallyfy.com/amit
