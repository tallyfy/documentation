The Tallyfy Manufactory collector allows you to send actor data and events, which can be viewed in our simple custom interface.

You can ingest data through a REST API, a websocket, or a CLI agent. Note that you can't ingest data before creating both actors and events schemas. Please refer to [Implementation Guide](/products/manufactory/implementation/).

The collector base URL is:

`https://collector.tallyfy.com/`
