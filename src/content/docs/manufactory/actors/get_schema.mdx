## Get an actor schema via API

You can get actors schema belonging to a specific project using the endpoint `GET /actors/schema/{org_id}/{project_id}`.

### Authentication

The following header is required for all API calls:
- `X-Manufactory-Auth` - Manufactory access token


### Required values

- `org_id` (required) string - Manufactory org ID.
- `project_id` (required) number - Manufactory project ID.


### Response example

```json
{
  "attributes": [
    {
      "name": "city",
      "display_name": "City",
      "data_type": "text"
    },
    {
      "name": "country",
      "display_name": "Country",
      "data_type": "text"
    },
    {
      "name": "email",
      "display_name": "Email",
      "data_type": "text"
    },
    {
      "name": "username",
      "display_name": "Username",
      "data_type": "text"
    }
  ],
  "org_id": "xyz",
  "unique_id": "email",
  "project_id": 12345
}
```
