## Understanding magic links

Magic links are specialized URLs that perform predefined actions in Tallyfy. They can pass data parameters and trigger specific system behaviors through URL manipulation.

:::caution[Authentication requirement]
You must be signed into Tallyfy to use magic links. Unauthenticated users will be prompted to sign in first.
:::

## Accessing magic links

Magic links are available in **Settings > Integrations > Magic Links**.

## Magic link capabilities

Magic links can:
* Create one-off tasks
* Launch processes
* Complete tasks
* Reopen tasks
* Add comments
* Edit deadlines
* Update form fields

## Creating magic links

### For one-off tasks

1. Click **Get a Magic Link**
2. Select **Create a One-Off Task**
3. Customize the URL parameters to set task attributes

### For process launches

1. Click **Get a Magic Link**
2. Select **Launch a Process**
3. Choose a template from your library
4. The system generates a link containing your organization ID and template ID

## Magic link structure

A typical process launch magic link with kickoff field data:

```
https://go.tallyfy.com/49b796c6b1e422181141ab801bfeb8bd/process/ab5c83b3d433798b571f29d1cfeec87f/create?default_run_name=Apr%2029,%2007:57am%20-%20Employee%20onboarding&amp;ko_fields=%255B%257B%2522employee-name-223217%2522:"Ruku Kadu"%257d%255d&amp;launchprocess=true
```

### Using auto-naming

To simplify magic links, you can use the auto-naming feature:

```
https://go.tallyfy.com/49b796c6b1e422181141ab801bfeb8bd/process/auto?default_run_name=Apr%2029,%2007:57am%20-%20Employee%20onboarding&amp;ko_fields=%255B%257B%2522employee-name-223217%2522:"Ruku Kadu"%257d%255d&amp;launchprocess=true
```

## Field value syntax reference

When passing form field values in magic links, use the following syntax:

|Field Type|Syntax Format|
|------------------|------|
|Short text box|`"short text box value"`|
|Long text box|`"long text box value"`|
|Dropdown|`{"id":1,"text":"1","value":null,"required":true}`|
|Checkbox|`[{"id":1,"text":"1","value":null,"required":true,"selected":true}]`|
|Radio button|`"1"`|
|Date/time|`"2021-04-29T00:00:00.000Z"`|
|File|`[{"id":"21b98bc82016581f1d32105502071083","url":"organizations/54339011418f12485ca7c5ba821cd585/file/21b98bc82016581f1d32105502071083/dl","source":"local","system":null,"step_id":null,"subject":{"id":"6b7d31b688f4bd738349767d40dff643","type":"Checklist"},"version":1,"filename":"social__5___1___4__jpg.jpg","uploaded_from":"ko_field"},{"url":"https://upload.wikimedia.org/wikipedia/commons/a/a3/June_odd-eyed-cat.jpg","source":"url","subject":{"id":"6b7d31b688f4bd738349767d40dff643","type":"Checklist"},"filename":"upload.wikimedia.org","uploaded_from":"ko_field"}]`|

## Additional magic link action examples

Magic links support various Tallyfy actions:

1. Create a one-off task:  
   `https://go.tallyfy.com/<YOUR-ORG-ID>/magic?action=createOotTask&default_task_name=Type in your task title`

2. Complete a task:  
   `https://go.tallyfy.com/<YOUR-ORG-ID>/magic?action=completeTask&task_id=b0d9153016723b3f6f54f152db430f09`

3. Reopen a task:  
   `https://go.tallyfy.com/<YOUR-ORG-ID>/magic?action=reOpenTask&task_id=b0d9153016723b3f6f54f152db430f09`

4. Write a comment:  
   `https://go.tallyfy.com/<YOUR-ORG-ID>/magic?action=writeAComment&task_id=b0d9153016723b3f6f54f152db430f09?comment=Default comment`

5. Edit task deadline:  
   `https://go.tallyfy.com/<YOUR-ORG-ID>/magic?action=editDeadline&task_id=b0d9153016723b3f6f54f152db430f09?deadline=2022-01-20 03:11:02`

6. Update form field:  
   `https://go.tallyfy.com/<YOUR-ORG-ID>/magic?action=updateFormFieldValue&task_id=b0d9153016723b3f6f54f152db430f09?form_fields={"d6410030b7fe633403688e900c5a6ba3":null}`

Magic links enable integration between Tallyfy and other systems, allowing one-click actions and data transfers without manual intervention.
