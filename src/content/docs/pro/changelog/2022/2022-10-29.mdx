### New Features

- **[API]** Added "equals any" option for automation rules on dropdown and checkbox fields, letting you check multiple values in one rule instead of creating separate rules for each option.
- **[API]** Guests can now easily recover their login access using a new "forgot password" link that includes simple verification for security.
- **[API]** Auto-Join automatically invites new signups from your company domain to join your existing organization instead of creating separate accounts.

### Bug Fixes

- **[API]** Tasks with only guest assignees now automatically include the process launcher, preventing them from disappearing from task lists.
- **[API]** Guest users now receive cleaner email notifications without redundant reminders after completing tasks in multi-step processes.
- **[API]** Fixed a rare error that could occasionally disrupt workflows, ensuring smoother and more reliable process automation.
- **[API]** Fixed an error that prevented checklists from loading properly in the Tasks view for all users.
- **[API]** Fixed custom organization logos now properly appearing in daily digest emails for a more personalized experience.
- **[API]** Fixed automation rules to properly validate required fields, preventing errors when creating "if this then that" rules.
- **[API]** Fixed search to now include archived processes in results, making it easier to find all your past work.
- **[API]** Fixed guest email validation to prevent errors when processing multiple guest invitations with invalid email addresses.
- **[API]** Added deletion tracking to activity feeds so you can see when templates and processes were removed for better audit trails.
- **[API]** Fixed folder organization bug that prevented setting a folder as its own parent, improving system stability and file management.
- **[API]** Fixed auto-joined users now getting the same helpful onboarding experience as invited users, ensuring everyone starts smoothly.
- **[API]** Guests now receive a single consolidated email when assigned multiple tasks instead of separate emails for each task, reducing inbox clutter.
- **[API]** Fixed outdated API documentation for creating processes to show correct data formats for different field types instead of just text strings.

### Security

- **[API]** **Security Enhancement:** Added name length limits and validation to prevent malicious redirects in email notifications while preserving legitimate names like "Dr. Smith".

### Performance

- **[API]** Guest forms with many fields now load faster, reducing wait times when external users access your processes.
- **[API]** We improved database connection efficiency to make Tallyfy faster and more responsive across all features.
