### New Features

- **[API]** You can now pin important templates to the top of folders, making them easier to find in long lists.
- **[API]** Super admins can now access any template across all organizations for better support troubleshooting.

### Bug Fixes

- **[API]** Fixed a bug that prevented checklists from being copied properly, so you can now duplicate your checklists without errors.
- **[API]** Fixed a database error that was preventing team groups from loading properly, ensuring smooth access to group management features.
- **[API]** Improved timezone handling to prevent unexpected application errors and ensure smoother workflow processing.
- **[API]** Fixed error when editing task assignments after deleting groups - tasks now automatically reassign to individual members instead of breaking.

### Security

- **[API]** Fixed search results showing restricted processes that users didn't have permission to view, ensuring better privacy and access control.
