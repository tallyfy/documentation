### New Features

- **[UI]** Added a toggle in Power Tools settings to control whether people get notified when you change task deadlines, giving you more control over notifications.
- **[UI]** We added "eagle view" - a new way to visualize procedure templates that clearly shows how automation rules connect steps and form fields together.
- **[UI]** Email tasks now show a helpful warning when you're the only assignee, preventing confusion since emails won't be sent to yourself.
- **[UI]** We created a dedicated testing environment that helps our team reproduce and fix bugs faster, leading to quicker resolution of issues you report.
- **[UI]** Document processes now let you complete tasks and create new linked tasks directly from the right pane, with better sorting and mobile-friendly design.

### Changes

- **[UI]** Launching document templates now skips the redundant description step, taking you directly to the customizable fields for faster workflow setup.
- **[UI]** Made Settings pages easier to read by adding proper spacing between section headings and cards.
- **[UI]** Made the "Hold tight..." text smaller and less distracting when launching processes to improve the visual flow.
- **[UI]** We completed comprehensive quality testing before releasing updates, ensuring a smoother and more reliable experience across all workflows.
- **[UI]** Settings navigation redesigned with an improved left sidebar structure, making it easier to find and access all configuration options.
- **[UI]** Added a colored left border to clearly show which task or step you've selected, making it easier to see what you're currently working on.
- **[UI]** Process sidebar now shows clickable template links, making it faster to jump back to the original template for reference.
- **[UI]** Redesigned the Snippets interface with a searchable, sortable table and improved editing modal for faster snippet management.
- **[UI]** Enhanced Google Analytics tracking to better correlate user activities across sessions, helping you get clearer insights into workflow usage patterns.
- **[UI]** Document editing is now simpler with all controls moved to the right sidebar, eliminating confusing mode switches while keeping full editing capabilities.
- **[UI]** You can now create new guests directly from the guest list with a "+ Create a new guest" button, and they instantly appear without refreshing your browser.
- **[UI]** Improved real-time updates so changes to processes and tasks now appear instantly in your browser without refreshing the page.
- **[UI]** We built enhanced analytics capabilities to better understand how you use workflows, helping us improve your experience based on real usage patterns.
- **[UI]** Organizations can now set custom default deadlines for template steps instead of being stuck with the fixed 5-day default.
- **[API]** Improved folder navigation in Favorites by showing the complete folder structure, making it easier to organize and find your templates.

### Bug Fixes

- **[UI]** Fixed @-mention search to instantly recognize the first letter of user names, making it faster to find and mention contacts.
- **[UI]** Fixed template edit menu options (Archive, Move, Duplicate) that were hidden behind other elements when folders contained many templates.
- **[UI]** Fixed intermittent page loading errors by optimizing email delivery settings and improving how templates load for organizations with many processes.
- **[UI]** Fixed form field options duplicating when switching between dropdown, checklist, radio, and table options while creating templates.
- **[UI]** Fixed dropdown menu delays in Tasks view so you can now reliably switch between "Tasks from processes" and "Tasks from blueprints" without waiting.
- **[UI]** Fixed large text fields with formatting not displaying properly when viewing approval processes, ensuring your rich content always appears correctly.
- **[UI]** Fixed video uploads in the editor so MP4 files now work properly alongside MOV and WEBM formats.
- **[UI]** Restored the missing border between the top navigation and content areas in the Admin Portal for clearer visual separation.
- **[UI]** Fixed template launch wizard unnecessarily showing blank description screens when HTML formatting made content appear empty to users.
- **[UI]** Fixed a technical error that was causing unexpected interruptions while using the app, making your workflow experience smoother and more reliable.
- **[UI]** Fixed a JavaScript error that was preventing certain forms from working properly, making the app more reliable.
- **[UI]** Fixed admin view of member profiles to correctly display team membership and timezone information instead of showing job titles in wrong fields.
- **[UI]** Fixed broken upgrade links during trials so you can now properly access billing options when hitting usage limits.
- **[UI]** Fixed glitches when adding guests to processes - button now activates only when email is entered and duplicate additions are handled smoothly.
- **[UI]** Fixed linked task deadlines to correctly use your organization's default deadline setting instead of an outdated 5-day default.
- **[UI]** Fixed an issue where a form field title was missing, making it easier to understand what information to enter.
- **[UI]** Fixed "Mine" filter in Tasks view to only show your assigned tasks after automations run, eliminating confusion from seeing everyone's tasks.
- **[UI]** Fixed tag filtering in Tracker to show only processes that actually contain the selected tag, making it easier to find what you're looking for.
- **[UI]** Standard users can now favorite templates without needing edit permissions, making it easier to bookmark your most-used processes.
- **[UI]** Fixed cropping issue where content was cut off at the top of pages when administrators were using masquerade mode to view as other users.
- **[UI]** Fixed missing notification labels when using Tallyfy in German and other non-English languages - notifications now display properly without needing page refreshes.
- **[UI]** Fixed the missing "Insert variables" button when automatically naming processes and removed confusing HTML formatting option in the template editor.
- **[UI]** Fixed confusing issue notifications for guests by clarifying that only organization members can resolve task issues.
- **[UI]** Fixed notification links for mentioned users so clicking them now properly opens the specific task, even when it's not assigned to you.
- **[API]** Fixed a behind-the-scenes error that could occasionally interrupt your workflow experience, making the platform more reliable and stable.
- **[API]** Email notifications for comments, mentions, and task updates now deliver reliably after resolving service configuration issues.
- **[API]** Group @mentions in comments now display the actual group name instead of confusing technical codes in notifications.
- **[API]** Fixed daily digest emails to respect your chosen schedule, so you now receive them only on your selected days of the week.
- **[API]** Fixed database setup issues that were preventing new team members from getting started quickly with development work.
- **[API]** Fixed folder deletion to properly clean up associations while keeping your processes safe and accessible when reorganizing your folder structure.

### Removed

- **[UI]** We streamlined our notification approach by using existing chat systems instead of adding bulk push notifications, keeping your interface clean and focused.

### Performance

- **[API]** Made backend deployments faster and more efficient by cleaning up unused system resources.
