### New Features

- **[UI]** Administrators can now set default visibility permissions for new templates, preventing test templates from cluttering your organization's template library.
- **[UI]** Process tags now display with custom colors on tracker cards and clicking them instantly filters to show matching processes.
- **[UI]** Added location tracking columns in member and guest management to show last access city and country, helping administrators monitor team activity and security.
- **[UI]** Added a notification banner system that automatically alerts you about scheduled maintenance and updates, so you'll always know about any planned downtime in advance.
- **[UI]** Approval steps can now automatically re-open any previous task when rejected, making it easier to handle revisions and corrections in your processes.

### Changes

- **[UI]** Streamlined document template launches by adding bulk assignment options and combining multiple task notifications into single emails, reducing repetitive steps and email clutter.
- **[UI]** Enhanced table customization options in document templates, including the ability to remove borders for cleaner formatting.
- **[UI]** Document reading now shows a table of contents on the left side for easier navigation and consistent styling across all viewing modes.
- **[UI]** Improved data handling for organization snippets to boost performance and ensure more reliable data processing.
- **[UI]** We completed comprehensive quality testing across the entire platform to ensure all recent fixes work perfectly before release.
- **[UI]** Added a dropdown menu to assignee pills showing email delivery status and quick actions like viewing tasks or removing assignees.

### Bug Fixes

- **[UI]** Fixed auto-join behavior to allow proper testing of new organization creation flows using email address variants.
- **[UI]** Fixed bulk assignment and deadline changes in Power Tools to properly apply to all tasks, including hidden ones.
- **[UI]** Fixed mismatched fonts in task completion settings to ensure clean, consistent text styling across all views.
- **[UI]** Improved spacing and layout in the Members section for organizations using Single Sign-On to make inviting new members look cleaner and more polished.
- **[UI]** Fixed confusing role creation flow, persistent login prompts, signup validation gaps, and missing task completion credits to make the platform smoother.
- **[UI]** Improved button styling in the Launch Process Wizard so all buttons display properly without compressed text.
- **[UI]** Fixed Blueprint Library search to work across all template types instead of being limited by the current filter selection.
- **[UI]** Fixed missing buttons, icons, and member information in Settings that weren't displaying properly, plus resolved search loading issues.
- **[UI]** Fixed white bar visual glitch in sidebar when administrators use masquerade mode for cleaner interface consistency.
- **[UI]** Fixed task viewing permissions so anyone can now see task details, while only assigned members can complete them.
- **[UI]** Fixed popup menus and help tooltips that were getting hidden behind other interface elements, making them easier to see and use.
- **[UI]** Fixed error handling when task titles exceed 600 characters, now showing clear guidance instead of confusing server errors.
- **[UI]** Fixed task assignee removal to only remove the selected group instead of clearing all assignees, preventing tasks from becoming inaccessible.
- **[UI]** Fixed table form fields losing data when switching pages or deleting rows, ensuring your work is properly saved.
- **[UI]** Fixed activity logs disappearing when switching between tabs in process settings, so you can now view all logged activities regardless of which tab you visited first.
- **[UI]** Fixed onboarding processes to properly show the congratulations screen when all tasks are completed, making the completion experience consistent across all process types.
- **[UI]** Fixed occasional issue where the Tasks view wouldn't show tasks assigned to other users when task restrictions were enabled.
