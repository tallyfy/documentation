### New Features

- **[API]** Added a "Team" field to user profiles so you can specify which team or division you belong to for better organization.
- **[API]** Your company logo now appears next to Tallyfy emails in Gmail and other supporting email services for better brand recognition.

### Changes

- **[API]** Cleaned up task assignment emails by removing unnecessary details and making bot-assigned tasks sound more professional for a better user experience.
- **[API]** API users can now filter snippets by title and sort by name or date for faster, more organized data retrieval.
- **[API]** Administrators can now set default deadlines for all users across their organization, making it easier to standardize process timing company-wide.
- **[API]** Tasks and processes can now be organized into multiple folders just like templates, making it easier to find and filter your work across custom views.

### Bug Fixes

- **[API]** Fixed a bug where certain templates would fail to auto-launch after step completion, ensuring all automation rules now work reliably.
- **[API]** Fixed Activity tab error that occurred when viewing checklists with automation rules, ensuring smooth navigation and tracking.
- **[API]** Process names now have a 300-character limit to prevent system errors and ensure reliable notifications.
- **[API]** Fixed table form fields not saving properly when created through our API documentation, eliminating validation errors.
- **[API]** Fixed a bug where duplicating steps with table form fields would fail with a validation error, making it easier to copy complex steps.
- **[API]** Fixed slow loading when viewing team member statistics - now loads in under 3 seconds for better performance.
- **[API]** Fixed API data processing to handle different input types more reliably, preventing errors and ensuring smoother performance across all workflows.
- **[API]** Fixed admin profile settings to correctly display and update the selected user's email preferences instead of the admin's own settings.
- **[API]** Fixed favoriting issue so you can now bookmark templates and processes without needing edit permissions.
- **[API]** Fixed migration reliability issues during project setup by replacing external dependencies with internal data sources, making the setup process more stable and consistent.
- **[API]** Fixed tag filtering in Tracker to only show processes that actually have the selected tag, making filtered results clearer and more predictable.

### Removed

- **[API]** Removed experimental signup protection that was incorrectly blocking legitimate users, ensuring smoother account creation for everyone.

### Performance

- **[API]** Improved system performance by optimizing our deployment infrastructure, resulting in faster response times across the platform.
