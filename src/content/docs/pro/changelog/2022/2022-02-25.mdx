### New Features

- **[UI]** Table headers now stay visible when scrolling through long lists in Tracker view, making it easier to navigate large datasets.
- **[API]** Administrators now get instant notifications when team members publish templates publicly, making it easier to review and maintain quality control.

### Changes

- **[UI]** Improved asset deployment speed by optimizing file handling and migrating to Cloudflare Pages for faster loading times.
- **[UI]** Email notifications from @mentions now link directly to the specific comment, making it faster to find and respond to important discussions.
- **[UI]** Enhanced system stability and reliability through comprehensive pre-release testing to ensure a smoother experience for all users.
- **[UI]** Fixed the empty tasks page to show a clearer "There's no tasks in this view" message and ensured all task type options display properly.
- **[UI]** Improved the Industry dropdown when publishing templates with better browsing, fixed spelling errors, and added new "Other" and "General" options.
- **[API]** We improved our internal development process to deliver features and fixes faster and more reliably.

### Bug Fixes

- **[UI]** Fixed slow loading times in the Public Library so you can browse and find templates much faster.
- **[UI]** Fixed Guest Tasks view to ensure external collaborators can properly access and manage their assigned tasks.
- **[UI]** Fixed right pane alignment issues in Tasks view that affected users on the Docs plan, ensuring consistent layout across all subscription types.
- **[UI]** Fixed missing "+Guest" icon in task lists that helps you quickly assign outside collaborators to tasks that need them.
- **[UI]** Fixed support admin interface to correctly display the actual count of public templates instead of always showing "0" when viewing search results.
- **[UI]** Fixed layout issues in Tasks view when using Masquerade mode so content displays properly without cutoffs or excessive spacing.
- **[UI]** You can now upload `.lic` license files directly in document editors and file upload fields without getting error messages.
- **[UI]** Fixed logo hover animation to maintain consistent size, creating a smoother and more professional visual experience.
- **[UI]** Fixed the "Add comment" button animation that would get stuck in a loop when clicked without entering text, making the interface smoother.
- **[UI]** Activity logs now appear instantly when you toggle the "Auto-name Process?" setting in templates, instead of taking 4 minutes to show up.
- **[UI]** Fixed strikethrough formatting so all completed tasks show consistently, regardless of how long the task title is.
- **[UI]** Fixed issue where removed team members were still getting task assignment emails through group memberships.
- **[UI]** Fixed group creation to work properly when adding only team members without any guest users.
- **[UI]** Fixed confusing print options that appeared when viewing templates in pop-up windows, since those views don't actually support printing.
- **[UI]** Clicking group icons in the organization members list now properly opens the group edit panel instead of redirecting to the main page.
- **[UI]** Fixed unresponsive "Add Another Option" and "Add New Column" buttons in the template editor, making form customization smoother and more reliable.
- **[UI]** Added a "Get Public Link" button to knowledge base forms, making it easier to find and share public forms with others.
- **[UI]** Fixed inconsistent task card heights in Tasks view by standardizing the appearance of one-off tasks to match regular tasks for a cleaner, more organized layout.
- **[UI]** Fixed one-off task descriptions disappearing after filling out forms, ensuring email tasks now send properly with all content intact.
- **[UI]** Template duplication now properly respects permissions, so only authorized users can create copies and reduce unwanted template clutter.
- **[API]** Fixed a rare database conflict that could occasionally prevent team group updates from saving properly.
- **[API]** Fixed duplicate email notifications so you now receive just one email when assigned a task instead of confusing multiple messages.
- **[API]** Group membership now properly reflects when organization members are disabled, so you won't see inactive users listed in groups anymore.
- **[API]** Fixed activity logging to properly show when you toggle the "Auto-name Process" setting instead of displaying outdated information.

### Removed

- **[UI]** We simplified folder organization by keeping template folders separate from process folders, making it easier to organize your work without confusion.

### Performance

- **[API]** Public library now loads much faster with improved response times, making it quicker to browse and find templates.
- **[API]** Fixed slow loading times in the Public Library by only fetching essential data, making the page load much faster for everyone browsing templates.
