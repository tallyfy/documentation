### New Features

- **[UI]** Admins can now control whether users can edit HTML code in text fields through a simple organization-wide toggle setting.
- **[UI]** Added task cloning to quickly duplicate tasks with all settings except comments, saving time when creating similar tasks.
- **[UI]** Added a visual progress bar to templates showing completion status (Started, In Progress, Almost Complete, Complete) so you can quickly see which templates need work without opening them.
- **[UI]** Added auto-naming for processes using required kickoff form fields, so administrators can create consistent process names automatically while still allowing manual overrides.
- **[API]** The watch feature now shows which team member set up monitoring on workflow items, making it easier to track accountability.
- **[API]** Guests can now watch and track specific tasks without needing a full account, making collaboration with external partners much easier.
- **[API]** Added the ability to clone any task and position it right after the original, making it faster to create similar tasks without starting from scratch.
- **[API]** Users can now watch processes, tasks, and other items to get notifications when things change, keeping everyone in the loop automatically.
- **[API]** Template updates now deploy more smoothly with controlled versioning that prevents editing conflicts and gives you flexible rollout options.
- **[API]** Added a new search endpoint that lets external tools quickly find specific tasks without downloading large amounts of data, making integrations faster and more reliable.
- **[API]** Added a configurable group size limit for task assignments to help organizations maintain manageable team coordination and performance.
- **[API]** Administrators can now create and manage groups, add or remove members and guests, and set group avatars for better organization.
- **[API]** Added Company Domains feature to help administrators track and manage domain names associated with your organization more effectively.
- **[API]** Users from the same company domain now automatically join the same organization when signing up, making team setup much easier.

### Changes

- **[UI]** We clarified that users with Launch permission can modify assignees when starting processes, even without Edit permission, as these are separate functions.
- **[UI]** Added "Why?" buttons to explain when tasks are restricted and suggest who else can complete them instead.
- **[UI]** Made the login timestamp in the header ribbon clearer by showing "Logged in 4 hours ago" instead of just the time, so you always know what the timestamp means.
- **[UI]** The Tracker view now loads faster and performs better with an upgraded table system for viewing your processes.
- **[UI]** Redesigned step setup to open with description editing first, making it faster and more intuitive to configure steps in your templates.
- **[UI]** Added detailed usage tracking to the template editor to better understand how you work and improve your experience based on real user behavior.
- **[UI]** Activity Feeds now show exactly what text changed in rich text fields, giving you a complete audit trail of content edits.
- **[API]** Template authors can now choose "immediate" or "approval-required" publishing when replacing live templates with draft versions for better workflow control.
- **[API]** Templates now support draft and published versions, letting you safely work on improvements without affecting your live processes.
- **[API]** New members now get helpful daily reminders for 18 days to complete their account setup, ensuring no one gets left behind during onboarding.

- **[UI]** Fixed guest form submissions now properly transfer data to member forms, ensuring seamless handoffs from public forms to internal workflows.

- **[UI]** Fixed an issue where step descriptions would sometimes not save when quickly closing the editing panel, ensuring all your text changes are now properly preserved.

### Removed

- **[UI]** Removed conflicting chat widget positioning that was causing interface issues, ensuring a cleaner and more reliable help experience.

### Security

- **[API]** Fixed a security issue where template steps could be accidentally deleted from the wrong templates, ensuring steps are only removed from their correct parent templates.
- **[API]** Enhanced task filtering security to ensure users only see tasks they're properly authorized to access when filtering by team member names.

### Performance

- **[API]** Database performance improved significantly with upgraded servers, making your workflows and processes load faster across the platform.
