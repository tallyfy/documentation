### New Features

- **[UI]** Administrators can now create and manage custom organization-wide roles in Settings, making it easier to assign the right people to tasks.

### Bug Fixes

- **[UI]** Form fields now save data more reliably when multiple people edit the same form at once, preventing any information loss.
- **[UI]** URLs in form field variables now automatically become clickable links when inserted into task descriptions, making it easier to access external resources.
- **[UI]** Fixed broken links in user removal screen so you can now click to quickly navigate to affected templates and processes in new tabs.
- **[UI]** Checklist answers now display vertically with line breaks instead of horizontally with commas, making form submissions much easier to read.
- **[UI]** Fixed form field toggle jumping around when clicked quickly, making it easier to set required fields without confusion.
