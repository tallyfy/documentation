### New Features

- **[UI]** Added a progress indicator in the template editor that guides you through setting up descriptions, assignees, and deadlines for each step.

### Changes

- **[UI]** Enhanced homepage experience for small trial teams by ensuring dashboard content displays properly regardless of team size.
- **[UI]** Added a visual notch indicator to highlight selected items in split-screen views, making it easier to see which item you're currently viewing.
- **[UI]** Tasks view now automatically opens the first task when you enter the page, making it easier to see task details right away.

### Bug Fixes

- **[UI]** Standard users no longer see confusing edit and delete buttons for groups they can't actually modify, creating a cleaner interface.
- **[UI]** Fixed overlapping dropdown and checkmark icons in member profiles to ensure clean, readable field displays.
- **[UI]** Settings sidebar now displays buttons and controls properly aligned with improved spacing and restored tag deletion functionality.

### Deprecated

- **[UI]** Added role management to Settings so you can now delete unused step assignee roles instead of having them clutter your lists forever.
- **[UI]** This proposed billing alert feature was evaluated and tabled pending further research, ensuring we build the right solution for payment notifications.

### Removed

- **[UI]** **Feature request for enhanced step privacy controls was consolidated into our new top-secret tasks feature for better data protection.**
