### New Features

- **[UI]** Added global styling controls for rich-text content, letting administrators customize fonts, colors and formatting across all processes from one central location.
- **[UI]** Added a "Give me ideas" button when naming new processes that shows helpful examples to make it easier to choose clear, specific names for your workflows.
- **[UI]** Standard users can now create and manage groups when enabled in Organization Settings, giving you more flexibility in team collaboration.
- **[UI]** Added a prominent "Share" button to tasks so you can easily send task links to team members and guests without hunting through hidden menus.
- **[UI]** Administrators can now pin important templates to the top of folders, making your most-used processes instantly visible to everyone.

### Changes

- **[UI]** Homepage template list now shows a better mix of pinned and recently updated templates to improve visibility of your latest work.
- **[UI]** Improved visibility of email digest controls so administrators can more easily manage notification settings for all members and guests.
- **[UI]** Removed unnecessary toolbar button from text editor to create a cleaner, more focused writing experience.
- **[UI]** Fixed text overflow in navigation menu for non-English languages by shortening translations to fit properly within available space.
- **[UI]** Improved deployment process reliability by resolving unnecessary browser preprocessing that was causing false error alerts during system updates.
- **[UI]** Clarified deployment workflow to ensure consistent and reliable updates reach users through our proven GitHub Actions system.
- **[UI]** Support staff can now view and access all organization users instead of being limited to just the first page of results.
- **[UI]** Enhanced our release process with more thorough quality testing before updates go live, ensuring better stability and reliability.
- **[UI]** Moved our app to faster Cloudflare infrastructure for improved loading speeds and better performance worldwide.
- **[UI]** Redesigned step settings to separate outbound and inbound integrations into dedicated sections, making webhook setup and external connections much clearer to configure.
- **[UI]** Launching processes now uses a cleaner step-by-step wizard that's easier to use on mobile and skips unnecessary screens automatically.

### Bug Fixes

- **[UI]** Fixed search to now include archived processes in results, making it easier to find and reference your completed work.
- **[UI]** Fixed a bug where creating processes would fail when tables contained empty values, ensuring smooth process creation regardless of table content.
- **[UI]** Fixed tooltip alignment when hovering over rule icons in process steps for a cleaner, more polished interface.
- **[UI]** Fixed the "Copy Link" button for sharing organization member links so it now properly copies to your clipboard instead of failing silently.
- **[UI]** Fixed app crashes when clicking highlighted areas that were causing endless loading loops, making the interface much more stable.
- **[UI]** Fixed trial plan switching so users can now easily upgrade or change their plan without validation errors blocking the process.
- **[UI]** Fixed missing email notifications when users are assigned tasks during process launch, ensuring you never miss important work assignments.
- **[UI]** Fixed rare error message that could appear during plan upgrades even when the upgrade completed successfully.
- **[UI]** Fixed notification display so you can now see who performed actions, including their names and profile pictures.
- **[UI]** Fixed user syncing errors by adding proper character limits for first and last names during account setup.
- **[UI]** Activity feeds now preserve your complete history even after deleting steps or other items from your processes.
- **[UI]** Duplicate button now works properly when copying steps in procedure templates, making workflow building faster and more efficient.
- **[UI]** You can now use required Date fields when auto-naming processes, so your process names include actual dates instead of being skipped.
- **[UI]** Fixed group permissions so Standard members can now properly access templates when assigned through groups instead of getting blocked with access errors.
- **[UI]** Fixed a bug where clicking in data tables would sometimes cause an error and disrupt your workflow.
- **[UI]** Fixed printing and PDF saving of templates from the configuration view to now capture the complete template instead of just the first page.
- **[UI]** Fixed printing of templates so all content now displays properly across multiple pages without cutting off text.
- **[UI]** Fixed publicized process URLs that were showing blank pages, so shared process links now load properly for everyone.
- **[UI]** Fixed form fields not showing up in Tasks view, so you can now properly fill out and complete tasks without validation errors.
- **[UI]** Fixed error that prevented guest users from viewing their assigned tasks when assignments were made through automation tools.
- **[UI]** Fixed text fields incorrectly showing "No information entered" even when you had typed content into forms.
- **[UI]** Fixed duplicate activity entries when launching processes so your activity feed now shows accurate counts and cleaner tracking history.
- **[UI]** Fixed "View HTML" button now properly appears for all users when HTML editing is enabled in organization settings.
- **[UI]** Fixed newly created Snippets to instantly appear in all snippet lists without needing to refresh your browser.
- **[UI]** Fixed blocker comment indicators not updating instantly across different views, so red borders and disabled buttons now appear immediately when issues are reported.
- **[UI]** Group logos now display properly throughout the app instead of showing just initials, making it easier to identify teams at a glance.
- **[UI]** Standard users can now delete tags and change tag colors just like administrators.
- **[UI]** Translation files now load faster thanks to improved caching that reduces loading times while keeping content fresh.
- **[UI]** You can now filter tags to show or hide automatically generated ones, with the setting staying put as you navigate between pages.
- **[UI]** Fixed left menu navigation on mobile and tablet devices to properly scroll and collapse, preventing cut-off elements and menu overlap issues.
- **[UI]** Fixed group permissions so that standard users can now properly complete tasks when they're part of an assigned group.
