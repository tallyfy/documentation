### New Features

- **[UI]** You can now upload CSV files directly in form fields and rich text editors, making it easier to share spreadsheet data in your processes.
- **[UI]** You can now set custom work weeks (4-7 days) for your organization so deadlines automatically avoid weekends and non-working days.
- **[API]** Improved developer onboarding process to deliver faster feature updates and bug fixes.

### Changes

- **[UI]** Enhanced our quality assurance process to catch and fix more issues before releases, ensuring smoother updates and better reliability.
- **[UI]** Moved default task deadline setting to a more logical location in Organization settings for easier configuration.
- **[UI]** Moved the default deadline setting from the branding section to organization details for easier discovery and more logical organization.
- **[UI]** Tasks can now be unlinked from processes more cleanly without comment history issues when relinking later.
- **[UI]** Renamed confusing "Settings" tabs in the template editor to "Account Settings" and "Template Settings" for clearer navigation.
- **[UI]** Enhanced our release process with comprehensive testing before deployment to ensure higher quality and reliability for all users.
- **[UI]** Guest users now see a cleaner, more professional interface with simplified empty-state screens that better reflect their role as external collaborators.
- **[UI]** Tables now display cleaner with improved column controls, smart navigation arrows that appear only when needed, and searchable dropdowns for easier data management.
- **[API]** Daily digest emails now load faster and more reliably by linking to your notifications instead of displaying them inline.
- **[API]** Templates are now automatically backed up in multiple formats whenever you make changes, so you can recover deleted templates if needed.
