### New Features

- **[API]** Templates now support explanatory videos from YouTube, Loom, and Vimeo to help teams better understand processes before launching them.
- **[API]** Added webhook notifications so you can instantly receive alerts when people you're watching complete tasks or launch processes.
- **[API]** Process watchers can now receive webhook notifications when tasks or processes complete, enabling seamless automation with your external systems.
- **[API]** Process watchers now receive automatic email notifications when tasks they're monitoring are completed and when entire processes finish.
- **[API]** Task watchers can now receive webhook notifications when watched tasks complete, giving you more flexible ways to stay updated on progress.
- **[API]** Webhooks now trigger when templates are replaced or removed, so you'll get automatic notifications when these workflows complete.
- **[API]** Guests can now watch tasks and processes to get notifications, making it easier for external collaborators to stay updated.
- **[API]** Added "Watching" feature that lets you follow specific team members and get notified when they complete tasks or launch processes, keeping you connected to important activities.
- **[API]** Process improvement suggestions from your team now automatically flow to template owners, making it easier to collect and review enhancement ideas in one place.
- **[API]** Added table form fields that let you create structured data with columns and rows, making it easier to collect and organize information in your processes.
- **[API]** Added custom workweek settings so deadline calculations now respect your organization's working days instead of defaulting to Monday-Friday schedules.

### Changes

- **[API]** Added direct SAML login URL to organization details, eliminating extra API calls for smoother SSO authentication.
- **[API]** Tasks now load faster and use less data by streamlining how we detect and display problem indicators.
- **[API]** Confirmed that users can only watch processes they have proper access to, maintaining secure permission controls.
- **[API]** Simplified login page with "Sign in" button and visual improvements to help more users get started faster.
- **[API]** Comments can now be filtered to show only user-typed messages, hiding automatic system notifications for a cleaner experience.
- **[API]** You can now "watch" tasks with the star button to get notified when they're completed, replacing the old favorites feature.
- **[API]** Replaced Favorites with a new "Watching" feature that sends smart notifications when tasks or processes you're watching are completed, helping you stay updated without overwhelming your inbox.
- **[API]** You can now set whether each step in your procedure templates assigns to the same person every time or changes based on conditions, giving you more flexible automation control.

### Bug Fixes

- **[API]** Fixed deployment issues that could occur when updating process tags, ensuring smoother system updates with better data handling.
- **[API]** Invitation links for withdrawn member invites now show helpful guidance instead of confusing error messages.
- **[API]** Fixed an error that prevented some templates from being properly restored after deletion, making template recovery more reliable.
- **[API]** SSO users clicking content links are now automatically redirected to complete authentication and then returned to their original destination seamlessly.
- **[API]** Fixed process status tracking so reopened kick-off forms correctly update from "completed" to active status for better workflow visibility.
- **[API]** Fixed email notifications for process ownership changes that were previously failing to send properly.
- **[API]** Fixed email notifications for task deadline changes to ensure they always send properly to keep you updated.
- **[API]** Fixed table formatting in email tasks so data now displays with proper spacing and professional styling, making your email communications much easier to read.
- **[API]** Fixed a behind-the-scenes error that was preventing comment threads from being created properly when handling rejected comments.
- **[API]** Fixed email notifications for task owners that were failing to send due to a template formatting issue.
- **[API]** Fixed search API documentation to correctly show GET requests with proper encoding for search terms containing spaces.
- **[API]** Fixed email notifications not sending when task deadlines were changed due to a deployment caching issue.
- **[API]** Fixed duplicate tag creation error that prevented users from properly organizing their processes with consistent tag names.
- **[API]** Fixed API filtering so you can now search for processes with multiple statuses at once, making it easier to find the exact processes you need.
- **[API]** Fixed a rare bug where the activity feed would occasionally fail to load due to missing data, ensuring consistent access to process updates.
- **[API]** Fixed system-generated comments to now appear in activity logs, giving you complete visibility into all automated actions across your processes.
- **[API]** Fixed task positioning in Tracker view so you can reliably reorder tasks without losing deadlines or metadata.
- **[API]** Fixed a crash that occurred when updating deadlines for child tasks, making deadline management more reliable.
- **[API]** Fixed signup and plan upgrade flows that were temporarily disrupted, ensuring smooth registration and account management for all users.
- **[API]** Automatic comments now appear correctly when tasks are assigned to groups, keeping everyone informed about task assignments.
- **[API]** Date variables in process forms now properly display in your organization's preferred date format instead of defaulting to a standard format.
- **[API]** Fixed the publicize toggle for archived processes - it's now properly disabled with a helpful tooltip instead of showing confusing error messages.
- **[API]** Fixed summary tables failing to display properly when containing certain field types, ensuring all data now renders correctly.
- **[API]** Fixed auto-assignment bug where task creators were incorrectly assigned to tasks that already had proper group members or other assignees.
- **[API]** Clicking email verification links multiple times now shows a helpful message instead of leaving you confused about what happened.

### Removed

- **[API]** We streamlined our notification system by removing redundant activity tracking that overlapped with existing unseen item indicators, keeping your workflow notifications clean and simple.

### Security

- **[API]** Enhanced security controls to ensure template steps can only be deleted from their correct parent templates, strengthening data protection.
