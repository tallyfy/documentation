### New Features

- **[UI]** Added a colorful top border to all pages that matches your organization's branding, creating a more polished and consistent visual experience.
- **[UI]** Added real-time translation of process content into your preferred language, so you can work seamlessly with templates and tasks created in different languages.
- **[UI]** Added "is any of" option for dropdown, radio, and checklist rules, letting you pick multiple values in one rule instead of creating separate rules for each choice.

### Changes

- **[UI]** Renamed form field labels from "Small Text" and "Large Text" to "Short Text" and "Long Text" for clearer content descriptions.
- **[UI]** Template filter in Tasks view now shows all related tasks, including those from launched processes, giving you complete visibility into template-based work.
- **[UI]** Form fields now show clearer open/closed indicators and removed confusing preview mode for a more intuitive editing experience.
- **[UI]** Renamed "Advanced" to "More Settings" in form field options to make configuration menus clearer and less intimidating.
- **[UI]** Document editing is now easier with improved drag controls for text fields, double-click renaming, and clear visual indicators for required fields.
- **[UI]** Simplified template permissions by combining two separate tabs into one, making it easier to control who can access your templates.
- **[UI]** Tasks can now be quickly edited with one click directly from the task view without entering full edit mode, making updates faster and smoother.
- **[UI]** Moved filters from sidebar to top of Tasks and Tracker views with removable filter pills, making it clearer what content you're viewing.
- **[UI]** Fixed broken reference links in workflows with clickable chips that show helpful details and let you easily reconnect or fix missing components.
- **[UI]** Enhanced task organization with new view options including form, Kanban, calendar, and table layouts to help you visualize and manage work the way that suits you best.
- **[API]** Improved system performance by removing outdated organization references from internal validation checks, making processes run more efficiently.
- **[API]** Login now shows smart suggestions when you use an email that's already registered, guiding you to the right sign-in method instead of showing confusing error messages.

### Bug Fixes

- **[UI]** Sign-out now works smoothly without getting stuck on a loading screen, taking you straight back to the login page.
- **[UI]** Fixed an issue where closing the step details pane would occasionally cause an endless loading screen, improving workflow reliability.
- **[UI]** Fixed headline formatting in documents so titles and headers now display with the correct font weight and style as intended.
- **[UI]** Fixed scroll arrow in template folders that incorrectly stayed visible when you reached the bottom of the list.
- **[UI]** Fixed spacing between sections in the workflow editor drawer to make editing steps cleaner and more visually organized.
- **[UI]** You can now delete blank form fields in procedure steps without having to name them first.
- **[UI]** You can now edit folders when filtering tasks, making it easier to organize and find your work.
- **[UI]** Fixed messy breadcrumb displays in the +Create menu for favorited templates nested deep inside multiple folders using smart truncation.
- **[UI]** Task assignee names now display clearly in a vertical list to better accommodate longer usernames and keep edit controls properly aligned.
- **[UI]** Form titles now correctly show the actual process name instead of the original template name, making it easier to identify which specific process you're working on.
- **[UI]** Made the scrollbar in the library section easier to grab and drag by increasing its width to match other scrollbars.
- **[UI]** Fixed a critical API issue during testing to ensure all new features and improvements work smoothly for users.
- **[UI]** Fixed template editor freezing when adding tags that match existing global tag names, making tag management smoother and more reliable.
- **[UI]** Fixed misleading scroll arrow in template library that incorrectly suggested more templates were available when you'd already reached the bottom.
- **[UI]** Fixed error when editing document templates that was preventing users from saving their changes successfully.
- **[UI]** Fixed templates showing permission errors when duplicating - now custom permissions are properly preserved and displayed correctly.
- **[UI]** Fixed auto-naming for processes to show clean text instead of messy HTML code when using date or template name variables.
- **[UI]** Fixed misleading comment entry fields in Public Process View that appeared editable but were actually read-only, improving clarity for external viewers.
- **[UI]** Improved signup flow to automatically redirect existing users to login with their email pre-filled, eliminating confusing error messages.
- **[API]** Improved staging environment stability by optimizing server configuration to ensure consistent performance during development testing.
- **[API]** Fixed OAuth2 redirect issue that was preventing external apps from properly connecting to Tallyfy after authorization.
- **[API]** Fixed API responses to show correct web addresses, preventing redirect loops and ensuring smoother navigation between features.

### Deprecated

- **[API]** Improved infrastructure reliability by updating to AWS's latest server management system for better performance and future compatibility.

### Removed

- **[UI]** Removed instructional video from template creation to streamline the process - you can now jump straight into building with built-in guidance.

### Performance

- **[UI]** Improved app performance by removing unnecessary background checks that were slowing down the system for all users.
