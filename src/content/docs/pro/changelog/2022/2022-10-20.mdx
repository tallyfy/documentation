### New Features

- **[API]** Standard and Light users can now create and manage groups when enabled by administrators, making team organization more flexible.

### Changes

- **[API]** Account status errors now show when your account was disabled and who disabled it, making it easier to understand and resolve access issues.
- **[API]** Improved error monitoring accuracy by filtering out routine authentication validation messages, making it easier to spot genuine issues.
- **[API]** API now supports retrieving organization signup data by masquerading as the admin user, making it easier to access setup information.
- **[API]** Tasks now show a clean activity log instead of cluttered bot comments, giving you better visibility into what happened without the mess.
- **[API]** Tasks View now shows tasks assigned to groups you're a member of, making it easier to see all your relevant work.

### Bug Fixes

- **[API]** Fixed template launching when form tables have empty columns - now processes start reliably even with blank table data.
- **[API]** Fixed a bug that prevented adding new guests to your organization, ensuring the guest invitation process now works smoothly.
- **[API]** Fixed API error that was preventing users from properly accessing their organization data and settings.
- **[API]** Fixed a bug where creating tasks with form fields would fail if no title was provided, making task creation more reliable.
- **[API]** Fixed billing system communication to ensure subscription updates and payment notifications are processed more reliably.
- **[API]** Enhanced activity history to keep all records visible even after checklists or other items are deleted, preserving your complete workflow timeline.
- **[API]** Fixed a search error that was preventing users from finding specific steps within their procedure templates.
- **[API]** Guest organization details now display correctly for all guest email addresses, improving visibility into external collaborator access.
- **[API]** Fixed SSO login issues when your organization changes email formats, ensuring seamless authentication with updated email addresses.
- **[API]** Fixed a backend error that could occasionally cause certain page loads to fail, ensuring smoother navigation throughout the app.
- **[API]** Fixed database queries to properly handle empty values, preventing errors when filtering tasks and processes.
- **[API]** Fixed a server error that could prevent certain secure links from working properly, ensuring reliable access to protected content.
- **[API]** Fixed a database issue that could prevent checklist updates from saving properly in certain situations.
- **[API]** Fixed group permissions so Standard members can now properly access templates and processes through the API without getting blocked by permission errors.
- **[API]** Fixed a system error that could occasionally disrupt your workflow experience, making the platform more reliable.
- **[API]** Fixed duplicate route names that were causing system errors, making the app more stable and reliable for all users.
- **[API]** Fixed email sender display to show "Tallyfy" instead of "postmaster" for clearer communication.
- **[API]** Fixed guest access issues so external collaborators can now reliably join and participate in your processes without errors.
- **[API]** Data processing now handles new column types more reliably, preventing errors when exporting workflow information.
- **[API]** Fixed email delivery issues by streamlining our email address format to ensure all task notifications reach you reliably.
- **[API]** Fixed slow loading when viewing large teams by breaking user lists into smaller chunks, making the experience much faster and more reliable.
- **[API]** Fixed slow task completion in large processes - completing tasks in complex templates now takes under 8 seconds instead of 40+ seconds.

### Security

- **[API]** Fixed error handling to prevent debug tools from running in production environments, improving overall system security.

### Dependencies

- **[API]** Upgraded our core framework to unlock faster search capabilities and improved email delivery options for all users.
