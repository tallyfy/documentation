### New Features

- **[UI]** You can now upload PDFs directly to create SOP templates without converting them first, making it faster to get your documents into Tallyfy.
- **[UI]** Proposed new automated step type that can fill forms and gather data using simple instructions, helping you complete routine web tasks without manual work.
- **[UI]** Added a new help dropdown menu with platform benefits, demo videos, and onboarding guides to make getting started easier.

### Changes

- **[UI]** Added bulk editing for procedure steps, letting you select multiple steps at once to add or remove milestones more efficiently.
- **[UI]** Removed inconsistent hover effects on integration cards to match the cleaner, streamlined design across all card views.
- **[UI]** Power Tools now lets you select specific tasks with checkboxes before updating deadlines, making bulk deadline changes much clearer and easier.

### Bug Fixes

- **[UI]** Fixed AI-powered process naming that was failing to generate creative names, so you can now get helpful suggestions when creating new processes.
- **[UI]** Fixed AI template generation getting stuck on loading screen - now creates templates instantly as expected.
- **[UI]** Templates now load faster and more reliably after resolving temporary delays that could cause blank screens or stuck loading states.
- **[UI]** Fixed a JavaScript error that could occasionally cause page elements to not load properly, making the app more reliable.
- **[UI]** Fixed slow loading in the Members Activity tab by addressing underlying errors that were causing delays.
- **[UI]** Fixed task summaries to properly display default values from dropdown and checklist form fields, ensuring all field types work consistently.
- **[UI]** Fixed misaligned dropdown arrows in member profile settings to improve the visual experience when viewing team member details.
- **[UI]** Fixed a client error that was preventing certain page updates from working smoothly, improving overall app reliability.
- **[UI]** Status indicators now display properly in a single line format for better readability when viewing your procedures and documents.
- **[UI]** Fixed PDF conversion issues in SOP Templates so all PDF formats now generate templates properly without the system becoming unresponsive.
- **[UI]** Fixed deadline display issues in process tracking views to show more accurate timing information.
- **[UI]** Fixed "Subject not found" errors when viewing tasks with job titles and restored missing bulk-assign job titles button in tracker view.
- **[UI]** Fixed page navigation in Tracker and Templates views to automatically scroll to the top when switching pages for better browsing experience.
- **[UI]** Fixed jarring page transitions in Action Plan view - now loads smoothly with consistent animations instead of flickering between blank pages.
- **[UI]** When setting up task presets, you can now see the actual names of assigned people and groups instead of confusing generic labels.
- **[UI]** Fixed filtering by groups in Tasks and Tracker views so you can now easily find processes assigned to specific teams.
- **[UI]** Rich text editor now smoothly scrolls down when typing long content, so you can always see your cursor position.

### Deprecated

- **[UI]** We're focusing our development efforts on native desktop and mobile apps rather than browser-based offline features to deliver better user experiences.
