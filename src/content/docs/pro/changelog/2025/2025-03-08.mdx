### New Features

- **[UI]** You can now upload PDFs directly to create SOP [templates](/products/pro/documenting/templates/) without converting them first, making it faster to get your documents into Tallyfy.
- **[UI]** Proposed new automated step type that can fill forms and gather data using simple instructions, helping you complete routine web [tasks](/products/pro/tracking-and-tasks/tasks/) without manual work.
- **[UI]** Added a new help dropdown menu with platform benefits, demo videos, and onboarding guides to make getting started easier.

### Changes

- **[UI]** Added bulk editing for procedure steps, letting you select multiple steps at once to add or remove milestones more efficiently.
- **[UI]** Removed inconsistent hover effects on integration cards to match the cleaner, streamlined design across all card views.
- **[UI]** [Power Tools](/products/pro/documenting/templates/how-can-power-tools-enhance-my-tallyfy-experience/) now lets you select specific [tasks](/products/pro/tracking-and-tasks/tasks/) with checkboxes before updating deadlines, making bulk deadline changes much clearer and easier.

### Bug Fixes

- **[UI]** Fixed AI-powered [process](/products/pro/tracking-and-tasks/processes/) naming that was failing to generate creative names, so you can now get helpful suggestions when creating new processes.
- **[UI]** Fixed AI [template](/products/pro/documenting/templates/) generation getting stuck on loading screen - now creates templates instantly as expected.
- **[UI]** [Templates](/products/pro/documenting/templates/) now load faster and more reliably after resolving temporary delays that could cause blank screens or stuck loading states.
- **[UI]** Fixed a JavaScript error that could occasionally cause page elements to not load properly, making the app more reliable.
- **[UI]** Fixed slow loading in the [Members](/products/pro/documenting/members/) Activity tab by addressing underlying errors that were causing delays.
- **[UI]** Fixed [task](/products/pro/tracking-and-tasks/tasks/) summaries to properly display default values from dropdown and checklist [form fields](/products/pro/tracking-and-tasks/tasks/what-are-form-fields-in-tallyfy/), ensuring all field types work consistently.
- **[UI]** Fixed misaligned dropdown arrows in [member](/products/pro/documenting/members/) profile settings to improve the visual experience when viewing team member details.
- **[UI]** Fixed a client error that was preventing certain page updates from working smoothly, improving overall app reliability.
- **[UI]** Status indicators now display properly in a single line format for better readability when viewing your procedures and documents.
- **[UI]** Fixed PDF conversion issues in SOP [Templates](/products/pro/documenting/templates/) so all PDF formats now generate templates properly without the system becoming unresponsive.
- **[UI]** Fixed deadline display issues in [process](/products/pro/tracking-and-tasks/processes/) tracking views to show more accurate timing information.
- **[UI]** Fixed "Subject not found" errors when viewing [tasks](/products/pro/tracking-and-tasks/tasks/) with job titles and restored missing bulk-assign job titles button in [tracker view](/products/pro/tracking-and-tasks/tracker-view/).
- **[UI]** Fixed page navigation in [Tracker](/products/pro/tracking-and-tasks/tracker-view/) and [Templates](/products/pro/documenting/templates/) views to automatically scroll to the top when switching pages for better browsing experience.
- **[UI]** Fixed jarring page transitions in Action Plan view - now loads smoothly with consistent animations instead of flickering between blank pages.
- **[UI]** When setting up task presets, you can now see the actual names of assigned people and groups instead of confusing generic labels.
- **[UI]** Fixed filtering by groups in [Tasks](/products/pro/tracking-and-tasks/tasks-view/) and [Tracker](/products/pro/tracking-and-tasks/tracker-view/) views so you can now easily find [processes](/products/pro/tracking-and-tasks/processes/) assigned to specific teams.
- **[UI]** Rich text editor now smoothly scrolls down when typing long content, so you can always see your cursor position.

### Deprecated

- **[UI]** We're focusing our development efforts on native desktop and mobile apps rather than browser-based offline features to deliver better user experiences.
