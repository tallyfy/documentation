### New Features

- **[UI]** You can now upload your existing Word or PDF documents to instantly create procedure templates using AI, saving hours of manual setup time.

### Changes

- **[UI]** Billing page now shows all information immediately without needing to click "Continue" and includes helpful links for discount codes.
- **[UI]** The task wizard now automatically disables conflicting options when launching processes from the same template, preventing confusion with clear visual cues and helpful tooltips.

### Bug Fixes

- **[UI]** Fixed unfilled form fields showing as green outlines when printing documents to PDF - now only completed fields appear for cleaner outputs.
- **[UI]** Fixed dropdown arrows that weren't responding to clicks across settings, Tasks view, Tracker, and template creation, making menus easier to access.
- **[UI]** Fixed the "TAKE OVER" button to consistently appear when you're part of a group assigned to a task, letting you claim group tasks instantly.
- **[UI]** Template favorite displays now properly update when team members are deactivated, removing outdated user names and fixing star highlighting.
- **[UI]** Fixed milestone task numbering to display correctly when milestones become empty, eliminating confusing "NaN" errors in your process view.
- **[UI]** Fixed the country dropdown arrow in billing settings to align properly and made the entire arrow clickable for easier selection.
- **[UI]** Fixed date calculations in forms and templates - existing date fields may need a quick manual update to display correctly across all views.
- **[UI]** Fixed duplicate tasks appearing in the Tracker view when using automations with milestones, ensuring each task shows up only once.
- **[UI]** Fixed form field prefix and suffix display to show correctly without unwanted line breaks and with proper spacing across all views.
- **[UI]** Fixed automation rules not updating when you modify and resubmit Kick-Off Forms, so your "if this then that" rules now work reliably every time.
- **[UI]** You can now edit roles on any task and use bulk role assignments through Power Tools with real-time status updates that work correctly.
- **[UI]** Process names are now properly limited to 300 characters when using auto-launch features, preventing errors and ensuring consistent naming across all workflows.
- **[API]** Fixed crashes when accessing processes containing multiselect fields set to empty via API, ensuring all your active processes and tasks remain accessible.
- **[API]** Fixed email delivery issues by aligning reply-to addresses with sender addresses to meet security standards and ensure emails reach recipients reliably.
