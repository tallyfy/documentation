### New Features

- **[UI]** You can now view and print any active process as a clean PDF document, making it easy to share progress updates or keep offline records.
- **[UI]** Document templates now include a form builder that lets you add fillable text and date fields directly in your documents, making them interactive when launched.
- **[API]** Guest users can now easily view all organizations they're part of through a dedicated API endpoint, improving their access to account information.
- **[API]** Template edit history now tracks which team member made the most recent changes, making collaboration clearer and accountability easier.
- **[API]** Added a new search engine to make finding your processes, templates, and tasks much faster and more accurate.
- **[API]** Enhanced automation tracking to lay groundwork for upcoming undo capabilities, making rule executions more transparent and reversible.
- **[API]** You can now tag tasks and steps to organize your work better, plus filter by tags to quickly find what you need.
- **[API]** Superadmins can now highlight the best public templates in the library, making it easier to find high-quality templates.
- **[API]** Anyone can now fill out your public forms without needing a Tallyfy account, making it easier to collect information from customers or external partners.

### Changes

- **[UI]** Clarified that Standard users can withdraw account invites but cannot disable existing members, which protects your team's account security.
- **[UI]** Made text easier to read throughout the app by adjusting font weight and color for a cleaner, more comfortable viewing experience.
- **[UI]** Steps with "show" rules now clearly display with gray backgrounds to indicate they'll be hidden until triggered, improving template editing clarity.
- **[UI]** Improved clarity of custom view filter options to help users better understand when to use "OR" vs "AND" conditions for more predictable results.
- **[UI]** Comments can now be tagged as "Blocker," "Improvement," or "Advice" with visual indicators to help organize feedback and streamline workflow management.
- **[UI]** Renamed "Deadline" to "Expiry date" in one-off task creation for clearer understanding of when tasks automatically expire.
- **[UI]** Organization settings tables now include sorting, filtering, and CSV export with improved display of member groups and dynamic counts in navigation.
- **[UI]** Organization switcher now clearly separates your member organizations from guest organizations, making it easier to navigate between different access levels.
- **[API]** Extended database backup retention to 35 days to better protect your data and provide more recovery options during monthly updates.
- **[API]** You can now fetch all form fields from a process in one API call instead of multiple requests, making integrations faster and simpler.
- **[API]** Added missing documentation for key API endpoints in our developer interface, making it easier to integrate with guest management and domain features.

### Bug Fixes

- **[UI]** Fixed unwanted empty space in the workflow editor that appeared next to steps, making the interface cleaner and more polished.
- **[UI]** Fixed commenting and member selection failures that occurred when team members didn't have complete profile names by automatically generating display names from email addresses.
- **[UI]** Fixed display issue in Action Plan view where task panels were misaligned and couldn't be scrolled or closed properly.
- **[UI]** Fixed visual indicators for deleted form fields in document templates so you can now clearly see which fields no longer exist.
- **[UI]** Fixed process cards showing up for guest users when all tasks were hidden, ensuring guests only see processes with tasks they can actually access.
- **[UI]** Fixed step panel dropdown behavior so clicking task type options no longer accidentally opens or closes step descriptions.
- **[UI]** Fixed group icons in Organization Settings so clicking them now properly shows all group members instead of only displaying guests.
- **[UI]** Fixed broken "Check status" link in Organization Settings that was preventing you from viewing email delivery status.
- **[UI]** Fixed a rare display issue where an archived process would incorrectly appear in the main view, preventing proper unarchiving or deletion actions.
- **[UI]** Fixed missing border between left and right panes in process view to match the intended design and improve visual clarity.
- **[UI]** One-off tasks now instantly appear in your Tasks view when created, eliminating the need to manually refresh or click "View task" first.
- **[UI]** Guest users no longer see misleading drag-and-drop handles on task cards, preventing confusion when they lack permissions to move tasks.
- **[UI]** Fixed rich text editor now appearing correctly when editing tasks through the left pane menu, ensuring consistent formatting options regardless of how you access edit mode.
- **[UI]** Email links to completed tasks now automatically switch your view to show completed items, so you can always see the task you clicked on.
- **[UI]** Process builder now responds to single clicks when switching between step description and form fields panels, eliminating the need for double-clicking.
- **[UI]** Fixed broken links in activity trails that previously showed "Page not found" when clicking on task comments, so you can now navigate seamlessly.
- **[UI]** Fixed form field deletion in procedure templates so you no longer see confusing error messages when removing fields after launching processes.
- **[UI]** Password reset links now show clearer instructions when expired, guiding you to request a new reset instead of showing confusing error messages.
- **[UI]** Improved member reassignment auto-suggestions to work more reliably when typing names, ensuring smoother team collaboration during task handoffs.
- **[API]** Fixed task linking to ensure tasks stay properly connected to templates even when template versions are updated.
- **[API]** Fixed an issue where archived processes with group-assigned tasks couldn't be permanently deleted, so cleanup now works smoothly.
- **[API]** File downloads now work properly from web browsers when accessing files through external integrations or embedded applications.
- **[API]** Fixed public template library API to properly return featured templates, making it easier to discover and access community-shared workflows.
- **[API]** Fixed confusing error messages when deleting form fields in templates - deletions now complete smoothly without requiring a page refresh.
- **[API]** Fixed confusing error messages that appeared when deleting steps from templates, even though the deletion was working correctly.
- **[API]** Fixed password reset to show clear error messages for expired links and proper confirmation when requesting new ones.
- **[API]** Fixed template tag removal to work smoothly without errors, especially for templates imported from our public library.
- **[API]** Fixed analytics data processing to ensure customer usage insights are properly captured and accessible in real-time dashboards.
- **[API]** Improved database performance monitoring to identify and resolve potential slowdowns before they impact your workflow experience.
- **[API]** Comments on checklist steps now properly appear in the activity feed, making it easier to track all feedback and discussions.
- **[API]** Activity feed now properly tracks when teams are created, updated, or deleted without showing duplicate entries.
- **[API]** User management actions like inviting and enabling users now appear in activity feeds so you can track team changes.

### Removed

- **[UI]** Removed confusing "All assignees must complete" option from email tasks since they don't show completion progress like other task types.

### Security

- **[UI]** Blueprint sharing security strengthened - unauthorized users can no longer view template content through direct links, protecting your sensitive processes.

### Performance

- **[API]** Database performance improved by optimizing indexes, making your workflows and process tracking load faster across the board.
