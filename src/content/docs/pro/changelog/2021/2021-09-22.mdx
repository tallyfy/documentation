### New Features

- **[API]** Simplified login process with email-first approach that automatically routes you to the right authentication method based on your account type.

### Changes

- **[API]** Improved error handling in step management to prevent rare edge cases that could interrupt your workflow.

### Bug Fixes

- **[API]** Fixed a bug where removing certain tags from templates would cause errors when accessing saved checklists, so tag management now works smoothly.
- **[API]** Fixed a bug where email notifications weren't being sent properly when assigning new people to one-off tasks.
- **[API]** Fixed tag deletion in Settings so you can now properly remove tags from templates and active processes without errors.
- **[API]** Fixed login issues for analytics dashboard access, ensuring authorized users can now securely view their permitted data without delays.
- **[API]** Fixed a bug that prevented the guest management page from loading properly when certain guest data was missing.
- **[API]** Fixed file storage organization so your team's assets are now properly grouped under your organization folder, making file management more reliable and organized.
- **[API]** Fixed search results to no longer show templates when searching for tags that have been deleted from your organization.
- **[API]** One-off task activity feeds now properly track all changes including title, deadline, and summary updates for better visibility.
- **[API]** Daily activity emails now display clean, readable text instead of messy HTML code when documents are updated.
- **[API]** Fixed "Page not found" errors when accessing archived processes in Tracker view, so you can now reliably view and search all your completed work.
- **[API]** Fixed Microsoft sign-in issues so you can now seamlessly log in to Tallyfy when already signed into Outlook in another browser tab.
- **[API]** Fixed an error message that incorrectly appeared when deleting special tags, even though the deletion was successful.
- **[API]** Fixed a rare file upload issue that could occasionally prevent documents from being properly saved to your processes.
- **[API]** Email verification links now work seamlessly across any device or browser, eliminating the frustrating redirect to login pages.

### Security

- **[API]** Enhanced security for faster and more reliable deployments by upgrading to secure HTTPS connections during our build process.

### Performance

- **[API]** Database performance significantly improved with optimized indexing, making pages load faster when viewing tasks and tracking processes.
