### New Features

- **[UI]** Added AI-powered step descriptions that automatically generate detailed task instructions with one click, making it faster to create comprehensive procedures.
- **[UI]** You can now see a complete history of changes made to any task, including who changed what and when, making it easier to track progress and stay informed.
- **[UI]** New signups with matching company email domains can now automatically join your existing organization instead of creating separate accounts.

### Changes

- **[UI]** Confirmed that checklist buttons intentionally blink when switching browser tabs to helpfully remind you where you left off.
- **[UI]** Added "Last updated" tracking at the process level so you can see when settings, owners, or permissions were modified.
- **[UI]** Replaced confusing switch buttons with clear tabs for navigating between workflow views, making it easier to switch perspectives.
- **[UI]** Streamlined the task editor form by removing duplicate "Add Form Field" buttons to make creating forms clearer and less confusing.
- **[UI]** Merged separate folder systems into one universal structure so you can organize templates, processes, and tasks together without confusion across different views.
- **[UI]** Improved app reliability and user experience through enhanced quality testing before releasing updates to ensure everything works smoothly.
- **[UI]** When you create a new folder in your Library, you're now automatically taken inside it so you can immediately start adding templates.
- **[UI]** AI template creation now generates both procedures and documents in 14 languages with faster streaming responses and clearer buttons.
- **[UI]** Simplified procedure page navigation by adding a table of contents in the left pane, making it easier to jump to specific tasks.
- **[UI]** Bot users are now hidden by default in the Members table to reduce clutter and make it easier to find your team members.
- **[UI]** Template names now automatically capitalize the first letter of each word for a more professional and consistent appearance.
- **[UI]** Added helpful guidance when previewing document templates to clarify that fields need activation before editing.

### Bug Fixes

- **[UI]** Fixed cursor jumping to the beginning when editing field names in documents, making text editing smoother and more predictable.
- **[UI]** Fixed template creation buttons that weren't responding when clicked, so you can now reliably create new templates from scratch or examples.
- **[UI]** Fixed "Due Soon" indicator in Tracker to correctly show processes due within 24 hours instead of showing those with more time remaining.
- **[UI]** Fixed layout distortion in the launch wizard where buttons and panels stretched incorrectly when navigating back or canceling.
- **[UI]** Enhanced AI auto-suggestions for process names to handle high-demand periods more gracefully while keeping workflows uninterrupted.
- **[UI]** Form field text colors now properly match your branding and no longer display unexpected green text when clicked or refreshed.
- **[UI]** Fixed an issue where users couldn't be removed from organizations when they had hidden task assignments, even after using task reassignment.
- **[UI]** WYSIWYG editor setting now stays enabled on document text fields when launching processes or switching between views.
- **[UI]** Fixed disappearing text when typing quickly in form fields, making data entry smoother and more reliable.
- **[UI]** Fixed comment filter dropdown arrow positioning in Tasks and Tracker views for better visual consistency.
- **[UI]** You can now combine multiple filter types in Tracker view to see exactly the processes you need, instead of filters replacing each other.
- **[UI]** Activity feed now shows accurate messages when you move templates between folders, making it clearer to track template organization changes.
- **[UI]** Mobile process launching now works smoothly with better text sizing, cleaner layouts, and easier-to-reach launch buttons that stick to your screen.
- **[UI]** Fixed Tracker view unarchiving to work smoothly without disruptive page reloads, matching the seamless archiving experience.
- **[UI]** Fixed template library settings panel now closing properly when switching between different views.
- **[UI]** Improved the create modal by fixing overlapping buttons and making the scroll arrow appear only when needed for a cleaner interface.
- **[UI]** Tasks accessed through email links now automatically show up regardless of your current filter settings, so you'll never land on a blank page again.

### Dependencies

- **[UI]** Enhanced text editor with improved copy-paste functionality for smoother document editing and formatting.
