### New Features

- **[API]** Administrators can now customize the "not found" error pages that users see, including personalized messages and animated GIFs for a better user experience.
- **[API]** You can now filter your Tasks view to see only tasks where you've been @mentioned by others, making it easier to find items that need your attention.

### Bug Fixes

- **[API]** Fixed API calculations to respect your organization's work week settings more accurately when retrieving default scheduling data.
- **[API]** Fixed missing process information in API responses when working with guests, making integrations more reliable and consistent.
- **[API]** Fixed a bug that could cause errors when viewing process details, making the interface more reliable.
- **[API]** Fixed global search to properly find and display tasks when searching by keywords, making it easier to locate any task quickly.
- **[API]** Fixed a crash that occurred when viewing processes that were previously owned by deleted users, ensuring the tracker loads smoothly for all processes.
- **[API]** Guest organization data now includes all available service information for more complete integration visibility.
- **[API]** Fixed subscription renewals to properly handle payment failures with grace periods instead of immediate termination.
- **[API]** Fixed template tags incorrectly showing in the public library, so now only relevant tags appear when browsing featured templates.
- **[API]** You can now upload .oft Outlook template files to your processes without any errors or restrictions.
- **[API]** Fixed Public Process View to properly show who completed tasks when the "Assignees" toggle is enabled, replacing confusing code numbers with actual user names.

### Performance

- **[API]** Tasks and processes now load 16x faster (from 16 seconds to 1 second) by streamlining data delivery.
