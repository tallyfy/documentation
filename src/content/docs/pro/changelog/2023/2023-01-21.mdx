### New Features

- **[UI]** Added new Assignee field type that lets you easily pick team members or guests with auto-complete when building forms and assignment rules.
- **[UI]** Added a variables picker when editing steps, so you can easily insert variables without manually copying and pasting them.
- **[UI]** AI-powered step suggestions now help you build procedure templates faster by recommending relevant workflow steps based on your template name or existing steps.
- **[UI]** We're building automated steps that complete themselves without human input, making your workflows even more hands-free and efficient.
- **[UI]** One-off tasks now support webhooks in the Advanced section, letting you automatically trigger actions in other apps when tasks are completed.
- **[UI]** Guests can now control their own email notification preferences, and admins can easily manage guest email settings from the Settings section.

### Changes

- **[UI]** Made email notification settings easier to scan by moving question mark tooltips to the left side of text for better alignment.
- **[UI]** We clarified how Blueprint folders (for organizing templates by department) and Process folders (for organizing by client) work separately to give you more flexibility.
- **[UI]** Fixed dropdown arrow alignment in Create Button for a cleaner, more polished visual appearance.
- **[UI]** Document Editor table of contents now shows correct labeling and excludes headings from blank spaces for cleaner navigation.
- **[UI]** After reviewing text contrast in our editor, we kept the current bold formatting as it provides sufficient readability while organizations can customize styling if needed.
- **[UI]** Clarified that the Launch wizard's Folders dropdown shows process folders (not template folders), so it appears empty when no process folders exist.
- **[UI]** Integrations menu now works consistently - bottom items like Magic Links and Import/Export no longer have scrolling issues when clicked.
- **[UI]** Made right panes displaying tasks and steps more visually polished with rounded corners across all views.
- **[UI]** Updated infrastructure to better support multiple product offerings and improve overall system reliability.
- **[UI]** Fixed visual inconsistencies with bullet points and spacing in the text editor caused by browser cache conflicts.
- **[UI]** Fixed image visibility in the Change Background modal so you can now see the full image preview without navbar interference.
- **[UI]** Library navigation now smartly shows folders in the main area when sections have few templates, making it easier to discover and organize your content.
- **[UI]** Kept password changes secure by requiring your current password when logged in, preventing unauthorized access if someone uses your device.
- **[UI]** Made the assignment box wider in Power Assign to use your full screen, making it easier to select members and guests.
- **[UI]** Enhanced quality assurance testing now catches more issues before releases, ensuring smoother updates and better reliability.
- **[UI]** You can now create and save custom filter presets in the Process Tracker, making it easier to find specific processes just like in the Tasks view.
- **[UI]** The +Create menu now opens with a click instead of hover, preventing accidental closures and making it easier to create new templates and processes.
- **[UI]** Redesigned left navigation with a "Favorites" section that reduces launching your most-used templates from four clicks to just one click.
- **[UI]** Organization switcher now shows all your workspaces at once in a cleaner grid layout instead of requiring you to click through multiple pages.
- **[UI]** Preview mode now shows tailored layouts for each template type - procedures display step overviews, forms focus on field completion, and documents maintain their structured format.
