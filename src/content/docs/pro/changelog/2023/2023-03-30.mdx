### New Features

- **[UI]** Improved SSO testing capabilities by setting up a dedicated production environment, making authentication more reliable for teams using company email accounts.
- **[UI]** Added "From TallyfyAI" tab that instantly generates complete procedure templates with steps, roles, and deadlines from just your process name and company type.
- **[UI]** Added colorful icons to help you instantly identify different types of steps like approvals, surveys, and e-signatures across your processes.
- **[UI]** New users with matching email domains now automatically join your existing organization instead of creating separate accounts, eliminating fragmented teams and duplicate setups.

### Changes

- **[UI]** Clarified that tracker presets already support "OR" filtering via the AND/OR toggle, letting you find processes matching any condition.
- **[UI]** Enhanced quality testing process now ensures smoother releases with fewer interruptions to your workflow experience.
- **[UI]** Template names in the library are now clickable and take you directly to the full view, making it faster to browse and open templates.
- **[UI]** Settings menu tabs now have clearer visual styling to make it easier to see which section you're currently viewing.
- **[UI]** Document templates now support faster Word document uploads with improved formatting accuracy, eliminating manual conversion steps.
- **[API]** Strengthened security evaluation process by reviewing advanced vulnerability scanning options to enhance our SOC-2 compliance and infrastructure protection.
- **[API]** Billing exceptions for organizations can now be managed instantly through admin settings instead of requiring code deployments and system downtime.

### Bug Fixes

- **[UI]** Fixed content display in procedure processes so text is no longer hidden behind the left panel in page view mode.
- **[UI]** Cleaned up unwanted hover effects in the Library activity feed for a smoother browsing experience.
- **[UI]** Fixed email formatting across the platform to prevent delivery failures and ensure all notifications reach users reliably.
- **[UI]** Fixed broken "Learn More" link in API trigger procedures to properly connect developers with our documentation.
- **[UI]** Fixed rare glitch that could accidentally create duplicate processes when completing tasks or filling fields in workflows.
- **[UI]** Fixed document content disappearing after page refresh in the template editor, so your work now saves reliably.
- **[UI]** Fixed document template field editing so you can now reliably add, remove, and rearrange fields without losing your work when refreshing the page.
- **[UI]** Fixed mobile document view navigation to use two rows instead of three, giving you more screen space to focus on your work.
- **[UI]** Document editor now automatically cleans up messy formatting when you paste content from external sources, preventing layout issues.
- **[UI]** Removed delete buttons from document template view mode to prevent accidental changes and improve clarity.
- **[UI]** Fixed process naming in magic links so custom names you set in the URL are properly applied when launching processes.
- **[UI]** Fixed document headers being cut off or unreadable when administrators use masquerade mode to view processes as other users.
- **[UI]** Fixed deep links to work properly with SSO authentication so users now land on their intended page instead of the homepage.
- **[UI]** Fixed form field labels appearing bold and blurred when expanding dropdown, checklist, radio, and table fields for smoother visual experience.
- **[API]** Fixed a data validation error that could prevent guest users from properly accessing shared checklists in certain situations.
- **[API]** New organizations now get a clean start with just one "Start here" template instead of confusing duplicates.
