### Changes

- **[UI]** Added visual handles to table headers so you can easily see and resize column widths across all pages.
- **[UI]** Template library menu now has consistent text and icon alignment, matching the clean layout you see in other parts of Tallyfy.

### Bug Fixes

- **[UI]** Fixed duplicate "Add Form Field" buttons in the task editor that were confusing users during form creation.
- **[UI]** AI-generated descriptions in templates now display completely without being cut off mid-sentence for better readability.
- **[UI]** Document comments now stay visible after refreshing the page, so your feedback and notes are always preserved.
- **[UI]** Fixed misaligned dropdown arrow in the comment filter to improve visual consistency in Tracker and Tasks views.
- **[UI]** Fixed visual glitches in document templates where checkmarks, spacing, and comments weren't displaying properly after launching.
- **[UI]** Tracker view now stays stable when deleting unused custom process view presets, only refreshing when absolutely necessary.
- **[UI]** Fixed dropdown menus in template folders that were incorrectly staying open when switching between different folders in your Library.
- **[UI]** Fixed tag filters in Tasks view to wrap to new lines when you have many tags, preventing them from covering up your task details.
