### New Features

- **[UI]** Added six new compliance cards including 21 CFR Part 11 for life sciences, web accessibility reporting, and real-time status monitoring for API and client performance.
- **[UI]** Process naming now gets smarter AI suggestions that appear instantly without slowing down the launch wizard, helping you create more meaningful process names.
- **[UI]** Added process linking to automatically share data between connected processes, eliminating manual copying of information across workflows.
- **[UI]** Processes now auto-archive after completion with a new setting that lets you choose when (1-90 weeks) to keep your workspace organized.
- **[UI]** Added a "Login Method" column to organization members list showing how each user can sign in (Native, Google, Microsoft, SSO) for better account management visibility.
- **[API]** You can now filter your tasks by specific due date ranges and view all matching results on one page without clicking through multiple pages.

### Changes

- **[UI]** Made task selection clearer by softening the right panel shadow so you can better see which task is currently selected.
- **[UI]** Kept dividers between task details for better readability and consistency across all views.
- **[UI]** Clarified Welcome Blueprint behavior to ensure new team members get onboarding processes while guests remain unaffected by automatic assignments.
- **[UI]** Task editing now auto-saves all changes instantly without needing save buttons, so you never lose work or wonder if changes stuck.
- **[UI]** Board view in Tracker now shows color-coded labels for each process type (documents, forms, procedures) making it easier to identify what you're tracking at a glance.
- **[UI]** Improved table column widths and added hover-based sorting indicators to make data tables easier to read and navigate.
- **[UI]** Verified that automation rules preventing circular task visibility work correctly with existing "show all tasks" safety feature.
- **[UI]** Password settings now only appear for users who signed up with email, eliminating confusion for those using Google or SSO authentication.
- **[UI]** Switched to Inter font across the entire interface for crisper text and better readability, especially for international characters.
- **[UI]** Short text fields now have a dedicated validation drawer with clearer options for email, URL, and length rules instead of basic number checks.
- **[UI]** Settings are now better organized with separate "Personalization" and "Customization" sections, making it easier to find the right options for your needs.
- **[UI]** Document navigation now has separate tabs for structure ("Navigate") and content elements ("Elements"), making it easier to find what you need while editing.
- **[UI]** You can now edit document fields directly on the page instead of needing a separate read-only view, making document editing faster and more intuitive.
- **[UI]** Deployments now happen during off-peak hours with improved sequencing to minimize service interruptions and keep workflows running smoothly.
- **[UI]** Search results now show folder paths and favorites to help you find important templates faster when browsing complex nested structures.
- **[UI]** Added a spreadsheet-style table view for editing procedure templates, making it faster to create and modify process steps in bulk.
- **[API]** Added creation dates to guest lists so admins can easily see when new members joined and sort by signup date.
- **[API]** Improved page loading speeds by implementing smart caching technology, making your workflows and processes load faster worldwide.
- **[API]** Added email tracking headers to better monitor when recipients open and click emails sent from your processes.
- **[API]** Added tracking identifiers to all emails to make it easier to troubleshoot delivery issues and improve support response times.
