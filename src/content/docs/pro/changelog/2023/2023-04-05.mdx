### New Features

- **[UI]** Added a live counter in the sidebar showing how many form fields are configured in your automation triggers, making it easier to spot and manage complex automations at a glance.
- **[UI]** You can now upload Word documents when creating document templates, and they'll automatically convert to HTML while preserving your formatting and images.
- **[UI]** Added visual icons for different step types (approval, email, payment, survey, etc.) throughout Tallyfy to make it easier to quickly identify what each task requires.

### Changes

- **[API]** Email reply addresses now use a consistent format across all notifications, making it easier to track and manage automated emails.

### Bug Fixes

- **[UI]** Fixed form fields not saving properly when launching processes, ensuring required field validation now works correctly.
- **[UI]** Fixed form field editing errors in document templates that prevented users from updating their information smoothly.
- **[UI]** Fixed deadline dates in the launch wizard to respect your preferred date format and display cleanly on one line.
- **[UI]** Fixed spaces in document field labels displaying as HTML code instead of normal spaces, making field names cleaner and easier to read.
- **[UI]** Fixed Library dropdown menus to automatically close when clicking elsewhere, preventing multiple dropdowns from staying open at once.
- **[UI]** Fixed procedure content being cut off by the navigation pane, so you can now read everything without manually collapsing the sidebar.
- **[UI]** Fixed unwanted hover effects in the Library activity feed for a cleaner, more polished interface experience.
- **[UI]** Comments now save and display instantly for all users, eliminating any delays or missing messages in real-time conversations.
- **[UI]** Fixed image editing in text editor so you can now resize and move images after placing them, eliminating duplicates and unwanted scrolling.
- **[UI]** Fixed tag count accuracy in Settings page to always show the correct number based on your current auto-generated tags setting.
- **[UI]** Fixed search results to no longer show templates when searching for tags that have been deleted from your organization.
- **[API]** Fixed a bug where inviting guests to templates would sometimes fail due to incorrect data validation, making guest collaboration more reliable.
- **[API]** Fixed form field updates to work more reliably by preventing duplicate submissions that could cause save errors.
- **[API]** Fixed duplicate quote marks appearing in the Activity Feed to show cleaner, properly formatted update messages.
- **[API]** Fixed a validation issue when creating recurring job types so the process now works smoothly without database errors.
- **[API]** Fixed an issue where new accounts were getting extra duplicate templates during signup, so now you'll only see the intended "Start here" template to get going quickly.
