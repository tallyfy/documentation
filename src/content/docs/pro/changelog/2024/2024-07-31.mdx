### New Features

- **[UI]** Onboarding templates now automatically launch for new team members, making it easier to get everyone up to speed consistently.

### Changes

- **[UI]** Released a major platform update with improved user interface design and enhanced performance across all core features.
- **[UI]** Added automation examples to help you design forms that prevent invalid step completions through better field requirements.
- **[UI]** You can now add new members directly to groups when inviting them, saving time by skipping the extra step after they join.

### Bug Fixes

- **[UI]** Fixed missing tooltips when hovering over process names in Tracker's Table view to match Board view behavior.
