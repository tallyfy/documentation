### New Features

- **[UI]** Support administrators can now selectively disable trial limits for specific organizations, allowing more flexibility for prospects who need extended evaluations.
- **[UI]** Light role members can now easily request account upgrades through helpful popup messages when they need to create or edit templates.

### Bug Fixes

- **[UI]** Fixed email digest settings where dropdown options showed confusing code text instead of readable day options, preventing changes from being saved.
- **[UI]** Fixed analytics status display in Settings > Integrations to correctly show when analytics is actually enabled, eliminating confusing "disabled" messages.
- **[UI]** Fixed email address variables in form fields that were showing duplicated characters when displayed in task descriptions.
- **[UI]** Fixed permissions not saving correctly when administrators left custom settings blank, ensuring admin access is properly maintained.
- **[UI]** Form field setup now prevents you from moving to other steps until dropdown, checklist, and radio button options are properly filled in.
- **[UI]** Fixed checklist form fields duplicating when switching between steps during setup, ensuring clean form configuration without unwanted extras.
- **[UI]** Fixed milestone positioning and step numbering to update instantly without needing page refreshes when editing templates.
- **[UI]** Tasks view tag filter now searches both individual tasks and entire processes with one click, making it easier to find what you need.
- **[UI]** Template duplication now copies webhooks and tags, so your automated workflows stay intact when you duplicate templates.
- **[UI]** CSV exports from Tracker tables now show correctly formatted data instead of garbled values like dates appearing as "01-May" for progress.
- **[UI]** Fixed Spanish translation errors and moved previously untranslated buttons and labels into Spanish, improving the experience for international users.
- **[UI]** Improved cursor appearance on dropdowns in Templates and Automations to clearly show when elements are clickable.
- **[UI]** Task menu now appears on completed tasks so you can easily copy guest links that properly show finished work.
