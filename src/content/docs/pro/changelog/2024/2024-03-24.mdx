### New Features

- **[UI]** Added a new FlowTable view that shows all your automation rules across every step in one color-coded table, making it easy to see how your workflows connect together.
- **[UI]** Added task status filtering to custom views so you can now filter processes by both who they're assigned to and whether tasks are active or completed.

### Changes

- **[UI]** Kept template duplication in settings menu to prevent accidental copying and maintain clean template organization.
- **[UI]** Improved AI procedure generation to create more comprehensive templates with multiple sections instead of single-section outputs.
- **[UI]** Enhanced table performance and functionality by optimizing our existing data grid component instead of switching to a new library.
- **[UI]** Template editor now shows the exact number of form fields in the drawer title, making it easier to see how many fields your template contains at a glance.
- **[UI]** Improved stability when updating checklist steps to prevent rare duplicate processing errors during template launches.
- **[UI]** Improved system reliability by investigating and resolving potential duplicate API call issues to ensure smoother workflow automation performance.
- **[UI]** Cleaned up confusing language by using "deactivate" consistently throughout the platform when changing member status, making it easier to follow instructions.
- **[UI]** Milestones now stay exactly where you place them in templates, keeping independent steps properly positioned between milestones instead of jumping to the top.
- **[UI]** Template editor now uses consistent "LAUNCH" button and clearer "SETTINGS" navigation, making it easier to launch processes and find configuration options.
- **[UI]** Added a simple checkbox interface for step dependencies that automatically creates the right automations behind the scenes, eliminating complex manual setup.
- **[UI]** Simplified adding steps to templates with a clearer "+" button that shows you exactly where new steps will be created.
- **[UI]** Tracker view now remembers your column and field customizations when you navigate away and return to the page.
- **[UI]** Streamlined process launching with fewer screens and simplified steps, making it faster to start processes especially on mobile devices.
- **[UI]** Template Editor now features a cleaner, simpler interface with inline editing that makes configuring process steps faster and more intuitive.

### Bug Fixes

- **[UI]** Fixed tasks incorrectly showing as "Completed Late" in the right pane when they were actually finished on time.
- **[UI]** Fixed URL redirects so that old bookmarked links with "blueprints" now automatically take you to the updated "templates" pages without breaking.
- **[UI]** Fixed a bug where removing yourself as the default assignee when creating out-of-office coverage would accidentally remove the wrong person instead.
- **[UI]** Fixed template folder organization to update instantly when moving templates between custom folders in the Templates Library.
- **[UI]** Fixed infinite loading when viewing form data in processes with many tasks, so you can now review all your submitted information without delays.
- **[UI]** Fixed the expand button on the templates folder sidebar so you can now easily collapse and expand your template folders.
- **[UI]** Restored right-click functionality to open templates, processes, and buttons in new tabs across Tracker, Homepage, and Templates Library for smoother navigation.
- **[UI]** Made Tasks, Tracker, and Templates views properly scrollable in all browser window sizes for better accessibility.
- **[UI]** Fixed table form fields not saving properly when filled out by guest users, ensuring all their responses are now captured correctly.
- **[UI]** Made template folder filters more obvious so you can easily see when you're viewing only documents vs all procedures.
- **[UI]** Fixed job titles with apostrophes now displaying correctly in the Task Editor interface.
- **[UI]** Fixed print preview for templates assigned to groups - now displays all template contents instead of just the first few pages.
- **[UI]** Fixed the "add step" button getting stuck in loading mode when editing document templates, so you can now smoothly add new steps to your workflows.
- **[UI]** Fixed document template editor panes to expand individually instead of both opening together, making the interface more predictable and consistent.
- **[UI]** Fixed document template editor so you can now close the left and right panels independently, giving you better control over your workspace layout.
- **[UI]** Fixed text field layout issue when adding new tasks or steps so the character limit alert no longer breaks the page design.
- **[UI]** Fixed jarring screen jumps when completing tasks by keeping the right panel smoothly open during transitions.
- **[UI]** Added milestone editing back to the Tasks view so you can now modify and create milestones directly from your task list.
- **[UI]** Fixed activity log links for standalone tasks so clicking them now correctly navigates to the right task instead of sending you to unrelated processes.
- **[UI]** Fixed heart reaction emojis displaying in the wrong color for users with specific browser or device configurations.
- **[UI]** Fixed email draft tasks to properly display the rich text editor when reopening task summaries multiple times.
- **[UI]** Fixed favorites counter to stay accurate when moving templates between folders - moving templates no longer accidentally removes them from favorites.
- **[UI]** Date fields now default to date-only instead of requiring time entry, making forms simpler and faster to complete.
- **[UI]** Fixed text visibility issue in Launch Wizard's bulk assignee search so keyword matches are now clearly readable with custom branding.
- **[UI]** Fixed task names showing as "null" in Table view column selection - now displays actual task names properly.
- **[UI]** Fixed missing "View Process" option in Tasks view menu, restoring quick navigation from individual tasks to their full process overview.
- **[UI]** Fixed table data from guest users now displaying properly in kick-off forms, ensuring submitted values appear correctly in all views.
- **[UI]** Fixed an issue where the "Bulk-assign job titles" tab showed up unnecessarily when launching templates that didn't actually use job title assignments, eliminating confusion.
- **[UI]** Removed jarring white flash when loading Tracker, Tasks, and Templates views for a smoother experience.
- **[UI]** Fixed misaligned "Please add a step" prompt when creating new procedure templates for better visual consistency.
- **[UI]** Fixed date and time values showing incorrectly in process names, ensuring accurate timestamps are displayed when launching templates.
- **[UI]** Process Status now shows "X Tasks Late" instead of "On time" when tasks miss deadlines, giving you clearer visibility into workflow delays.
- **[UI]** When you duplicate a template, it now appears in the same folder as the original instead of landing in "Uncategorized" for easier organization.
- **[UI]** Fixed confusing save buttons in user preferences - now all settings use one simple "Save Changes" button instead of mixing auto-save with individual save buttons.

### Deprecated

- **[UI]** Added bookmark functionality to help you quickly find important documents in your Library without needing separate filter presets.

### Removed

- **[UI]** Fixed form launching by removing the unnecessary step of naming forms, making it faster to get started.
