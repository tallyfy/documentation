### New Features

- **[UI]** Added a new "System Variables" section in the Variables dropdown that displays task and process IDs, making it easier to track workflows and build integrations.

### Changes

- **[UI]** Redesigned the step editor with organized tabs instead of accordion menus, making it faster and easier to configure your process steps.

### Bug Fixes

- **[UI]** Fixed sign-out button to properly redirect you to the sign-in page instead of getting stuck loading forever.
- **[UI]** Fixed template creation and step reordering issues discovered during quality testing to ensure a smoother experience.
- **[UI]** Fixed auto-archive process settings to properly save your custom timeframe instead of reverting back to the default 52 weeks.
