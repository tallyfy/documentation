### New Features

- **[UI]** You can now choose between instant or daily billing notification emails to reduce inbox clutter while staying informed about subscription changes.

### Changes

- **[UI]** Fixed automation rules to prevent selecting the same step in status actions that appears in the IF condition, eliminating potential conflicts.
- **[UI]** Clarified how "Linked to" fields work in completed tasks - they only redirect when viewed from Tasks, not when you're already in the target process.
- **[UI]** Improved conversion tracking on signup pages to better measure marketing performance and optimize user acquisition efforts.
- **[UI]** Made rejection notifications clearer by changing the confusing "something was missing" message to simply "Approval status updated."
- **[UI]** Fixed template permission errors so Standard users with edit access can now properly modify template settings without getting blocked by access errors.
- **[UI]** Added "Template Maintenance" and "Processes from Template" filters in Tasks view to help you find the exact tasks you're looking for more easily.
- **[UI]** Improved automation setup by making assignment actions clearer with better labels like "Add Assignees" and "Replace Assignees" so you know exactly what each option does.
- **[UI]** Updated member privilege labels in the admin panel to clearly show each user's actual role (Admin, Standard, or Light) for better clarity.
- **[UI]** Simplified billing display to show both full and light members under one Pro plan with accurate pricing for each role type.
- **[UI]** Made the entire logo upload area clickable and improved button styling, making it easier to change your organization's logo without confusion.

### Bug Fixes

- **[UI]** Fixed intermittent connection issue that temporarily prevented AI template generation and importing existing templates from working properly.
- **[UI]** Fixed a temporary glitch where certain template settings like description and tags couldn't be saved in the editor.
- **[UI]** Fixed date format options being cut off on mobile devices in Settings, so you can now properly select your preferred date format on any device.
- **[UI]** Fixed Tasks view filters to show archived tasks and clarified empty results, making it easier to find all your completed work.
- **[UI]** Email logs now show complete delivery status including failed and bounced emails for better visibility into your process notifications.
- **[UI]** Fixed infinite loading issue so guests can now properly access their assigned processes when navigating from Tasks or Tracker views.
- **[UI]** Fixed infinite loading issue in Templates Library folders so you can now browse examples and templates without interruption.
- **[UI]** Fixed processes failing to load when shared URLs contained active sidebar forms, ensuring smooth access for all recipients.
- **[UI]** Fixed missing divider line between "Started" and "In Progress" status options when using templates on iPad in landscape mode.
- **[UI]** Improved app stability and user experience through enhanced quality assurance testing before release.
- **[UI]** Fixed Microsoft login issues by updating our authentication system to work with Microsoft's new Entra Identity service, ensuring seamless signup and login.
- **[UI]** Fixed duplicate permission assignments in templates - now when you add a group, individual users from that group are automatically removed to keep permissions clean and simple.
- **[UI]** Improved admin controls for managing team members with clearer button labels, better confirmation prompts, and more reliable password reset functionality.
- **[UI]** Made task action buttons always visible without scrolling, so you can complete, approve or reject tasks instantly from any screen size.
- **[UI]** Fixed task duplication to properly copy deadline and permission settings, making it easier to create similar tasks without losing important configurations.

### Performance

- **[UI]** Guest loading is now faster with a streamlined data delivery system that reduces wait times.
- **[UI]** Improved loading speed across the interface by optimizing how we fetch team information in the background.
- **[UI]** Fixed slow loading in process tracking by optimizing how data refreshes, making the interface much faster and more responsive.
