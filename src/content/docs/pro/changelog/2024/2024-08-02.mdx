### Changes

- **[UI]** Added new automation examples to help you prevent task completion based on form field values by restructuring forms with required fields and valid options.
- **[UI]** You can now assign new members to specific groups directly when inviting them, so they instantly get access to relevant tasks when they join.

### Bug Fixes

- **[UI]** Fixed Tracker view loading error caused by outdated filter settings, restoring access for users experiencing crashes.
- **[UI]** Fixed missing tooltips when hovering over process names in Tracker's Table view for consistent experience across both viewing modes.
- **[UI]** Favorites menu now automatically removes archived or deleted items instead of showing confusing blank entries.
- **[UI]** Fixed missing process names in Guest Tracker view and prevented permission errors when guests click on template links.
- **[UI]** Updated navigation icons to prevent the chatbox from unexpectedly opening when accessing help resources, improving overall navigation clarity.
- **[UI]** You can now insert variables into default content for form fields, making it easier to pre-populate fields with dynamic information.
