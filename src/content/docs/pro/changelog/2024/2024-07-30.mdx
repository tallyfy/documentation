### New Features

- **[UI]** Organizations can now set a custom onboarding template that automatically launches whenever new members join, streamlining team setup.
- **[UI]** Added star/favorite buttons throughout the platform so you can now bookmark templates and processes from any view with changes syncing everywhere instantly.

### Changes

- **[UI]** Enhanced system stability and reliability through comprehensive quality testing before releasing new features and improvements.
- **[UI]** Upgraded our build system to improve deployment speed and reliability through enhanced caching capabilities.
- **[UI]** Search now automatically shows the most relevant results based on where you're searching from, so you'll find what you need faster without switching filters.
- **[UI]** Email Draft and Email Auto-Send tasks now automatically send a copy to the sender, so you can track what emails were sent from your processes.

### Bug Fixes

- **[UI]** Fixed member data exports to Excel and CSV to only include relevant columns, matching the clean format already available in PDF exports.
- **[UI]** Fixed the Cancel button in job title creation to show proper text instead of code snippets for a cleaner interface.
- **[UI]** Fixed broken API documentation links in Integrations and renamed "Settings" button to "Explore" for clearer navigation.
- **[UI]** Fixed tooltip alignment when hovering over process cards in Tracker Board view for better visibility and user experience.
- **[UI]** Fixed date formatting in Tracker Table view to properly display "Start Date" and "Last Updated" columns according to your personalized date format preferences.
- **[UI]** Fixed an issue where guests couldn't see referenced templates they had permission to view, ensuring all authorized content now displays properly.
- **[UI]** Fixed template editor display issues so group assignees now show up properly in all views without needing to refresh the page.
- **[UI]** Fixed task creation buttons to only activate after entering a name, and new tasks now appear instantly without needing to refresh the page.
- **[UI]** New users can now complete account setup smoothly even when their organization has reached task assignment limits.
- **[UI]** Fixed form layout issues in document and procedure templates by replacing text with icons, giving you more space to view and edit your content clearly.
- **[UI]** Fixed dropdown menus and toggle buttons in the task editor that weren't displaying properly on mobile devices.
- **[UI]** Fixed clicking on titles in the Favorites menu to properly navigate you to member details, templates, processes, and tasks for quicker access.
- **[UI]** Fixed an issue where you couldn't freely move steps between different milestones when editing procedure templates, giving you complete flexibility in organizing your workflow.
- **[UI]** Alert notifications now automatically disappear after 5 seconds instead of staying on screen forever, creating a cleaner experience when multiple alerts appear.
- **[UI]** Fixed document viewing for guest users by showing only their assigned tasks and removing unnecessary buttons, making the experience cleaner and more focused.

### Performance

- **[UI]** Improved user search performance for large organizations by implementing on-demand fetching, eliminating browser storage limits and speeding up results.
