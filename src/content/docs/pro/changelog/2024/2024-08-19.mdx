### New Features

- **[API]** New members now get automatically onboarded with your chosen template when they join your organization.
- **[API]** Added organization-level control to enable or disable generative AI features, helping enterprises meet their AI compliance policies.

### Changes

- **[API]** Improved API documentation clarity around process filtering options to help developers better understand existing capabilities without needing additional parameters.
- **[API]** Cleaned up development environment to ensure smoother code deployments and faster feature releases.

### Bug Fixes

- **[API]** You can now remove custom group logos and return to default initials display, giving you full control over your group's visual appearance.
- **[API]** Fixed the template library to automatically show the creation dialog for trial accounts with only one template, making it faster to get started.
- **[API]** Fixed a rare error that could occur when processing notification watchers for deleted objects, ensuring notifications work more reliably.
- **[API]** Guest task filtering now works correctly, showing only relevant tasks assigned to guests instead of displaying all tasks.
- **[API]** New users now automatically get their correct local timezone instead of defaulting to Chicago time, making your experience more accurate from day one.
