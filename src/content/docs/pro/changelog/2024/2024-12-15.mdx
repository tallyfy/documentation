### Changes

- **[UI]** Added detailed [API](/products/pro/integrations/open-api/) examples for updating different field types (text, date, dropdowns, checkboxes) to help developers integrate more easily.
- **[UI]** We're conducting comprehensive testing before our next major release to ensure the highest quality experience for all users.
- **[UI]** You can now filter [tasks](/products/pro/tracking-and-tasks/tasks/) by "All tasks" to see both active and completed work, or "Archived" to view tasks from archived [processes](/products/pro/tracking-and-tasks/processes/).
- **[API]** Added "All tasks" and "Archived" filters to help you view completed work and archived [processes](/products/pro/tracking-and-tasks/processes/) more easily.
- **[API]** Enhanced [API](/products/pro/integrations/open-api/) protection by implementing smarter rate limiting that tracks individual user activity, preventing service disruptions and ensuring fair usage for everyone.

### Bug Fixes

- **[UI]** Fixed temporary disruption to AI-powered [template](/products/pro/documenting/templates/) creation and step suggestions - all AI features are now working normally again.
- **[UI]** Fixed rare display issue where PDF links in rich text fields appeared missing after saving, ensuring all uploaded file links remain consistently visible.
- **[UI]** Dropdown menus in sub-navigation now stay visible when [task](/products/pro/tracking-and-tasks/tasks/) details panes are open, improving navigation flow.
- **[UI]** Fixed editor display issues in [document templates](/products/pro/documenting/documents/), making the guidance text editor cleaner and menus properly visible.
- **[UI]** Fixed dropdown menus, toggles, and labels that were cut off or misaligned in the [Templates](/products/pro/documenting/templates/) panel on mobile devices.
- **[UI]** Fixed truncated field names in tooltips when adding [variables](/products/pro/documenting/templates/variables/) to task descriptions, so you can now see the complete field name on hover.
- **[UI]** Fixed archived [processes](/products/pro/tracking-and-tasks/processes/) not opening properly, so you can now access all your archived tasks in the new [Tasks view](/products/pro/tracking-and-tasks/tasks-view/) filter.
- **[UI]** Fixed magic link launches that were incorrectly failing due to hidden formatting affecting [process](/products/pro/tracking-and-tasks/processes/) name character limits.
- **[UI]** Fixed rendering issue where rich text formatting and emoticons in field descriptions now display properly instead of appearing blank.
- **[UI]** Fixed AI step suggestions in [procedure templates](/products/pro/documenting/templates/) so you get fresh, helpful ideas when building your workflows instead of seeing no suggestions or the same ones repeatedly.
- **[UI]** [Form field](/products/pro/tracking-and-tasks/tasks/what-are-form-fields-in-tallyfy/) validations now work correctly for each individual field instead of accidentally copying rules between different fields.
- **[UI]** Fixed [guest](/products/pro/documenting/guests/) email validation in assignee fields so you can now assign [tasks](/products/pro/tracking-and-tasks/tasks/) to people outside your organization without needing to include internal team [members](/products/pro/documenting/members/).
- **[UI]** [Form field](/products/pro/tracking-and-tasks/tasks/what-are-form-fields-in-tallyfy/) updates now display instantly for [guests](/products/pro/documenting/guests/) without needing to refresh the page.
- **[UI]** Fixed [automation](/products/pro/documenting/templates/automations/) dropdown menus to prevent selecting the same step twice, making "if this then that" rules clearer and less confusing to set up.
- **[UI]** Fixed a technical error that occasionally prevented the help [support](/products/pro/miscellaneous/support/how-can-i-contact-tallyfys-support-team/) widget from loading properly for some users.
- **[UI]** AI-generated step descriptions in [templates](/products/pro/documenting/templates/) now display completely without cutting off mid-sentence, giving you fuller guidance for each step.
- **[UI]** Fixed public form inviter selection to show "Select user" instead of code when the previously chosen user is deactivated.
- **[UI]** Fixed file uploads so deleted files now disappear immediately without needing to refresh the page when reopening [tasks](/products/pro/tracking-and-tasks/tasks/).
- **[UI]** Fixed [Tasks view](/products/pro/tracking-and-tasks/tasks-view/) to automatically show all sections expanded, so you can see your tasks immediately without clicking to open each section.
- **[UI]** Fixed misleading "Saving changes" indicator that appeared when navigating [document templates](/products/pro/documenting/documents/) with [form fields](/products/pro/tracking-and-tasks/tasks/what-are-form-fields-in-tallyfy/), even when no actual changes were made.
- **[UI]** Fixed starred items showing incorrectly - favorites now sync properly between different pages so you'll see the right status everywhere.
- **[UI]** Fixed the [tracker view](/products/pro/tracking-and-tasks/tracker-view/) getting stuck on an endless loading screen when opened from a process page.
- **[UI]** Fixed column labels and dropdown indicators in AI-generated [templates](/products/pro/documenting/templates/) to make milestones and steps easier to navigate and understand.
- **[UI]** Fixed task completion button now correctly appears disabled for unassigned users in duplicated [tasks](/products/pro/tracking-and-tasks/tasks/), preventing confusion about permissions.
- **[UI]** Fixed dropdown arrows to show proper pointer cursor and improved click behavior for easier navigation.
- **[API]** Fixed complete button incorrectly appearing enabled for users without permission to complete restricted [tasks](/products/pro/tracking-and-tasks/tasks/).
- **[API]** Duplicated sub-tasks now appear correctly instead of staying permanently hidden, ensuring you can see and work on all your [tasks](/products/pro/tracking-and-tasks/tasks/).
- **[API]** [Template](/products/pro/documenting/templates/) duplication now correctly preserves the order of milestones, ensuring your duplicated templates maintain the exact same sequence as the original.
- **[API]** You can now search for groups by name instead of scrolling through the entire list.
- **[API]** Public forms now automatically stay accessible when the person who originally shared them is deactivated, ensuring uninterrupted form submissions.
- **[API]** Fixed an issue where some previously favorited [templates](/products/pro/documenting/templates/) and [processes](/products/pro/tracking-and-tasks/processes/) weren't appearing in your Favorites menu after our recent upgrade.

### Deprecated

### Security

- **[UI]** Fixed public form sharing to automatically switch ownership when an inviter is deactivated, keeping your forms accessible while maintaining security.

### Performance

- **[UI]** Fixed slow loading and freezing issues when viewing large tables and downloading files, making the app much more responsive.
