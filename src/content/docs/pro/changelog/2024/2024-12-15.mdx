### Changes

- **[UI]** Added detailed API examples for updating different field types (text, date, dropdowns, checkboxes) to help developers integrate more easily.
- **[UI]** We're conducting comprehensive testing before our next major release to ensure the highest quality experience for all users.
- **[UI]** You can now filter tasks by "All tasks" to see both active and completed work, or "Archived" to view tasks from archived processes.
- **[API]** Added "All tasks" and "Archived" filters to help you view completed work and archived processes more easily.
- **[API]** Enhanced API protection by implementing smarter rate limiting that tracks individual user activity, preventing service disruptions and ensuring fair usage for everyone.

### Bug Fixes

- **[UI]** Fixed temporary disruption to AI-powered template creation and step suggestions - all AI features are now working normally again.
- **[UI]** Fixed rare display issue where PDF links in rich text fields appeared missing after saving, ensuring all uploaded file links remain consistently visible.
- **[UI]** Dropdown menus in sub-navigation now stay visible when task details panes are open, improving navigation flow.
- **[UI]** Fixed editor display issues in document templates, making the guidance text editor cleaner and menus properly visible.
- **[UI]** Fixed dropdown menus, toggles, and labels that were cut off or misaligned in the Templates panel on mobile devices.
- **[UI]** Fixed truncated field names in tooltips when adding variables to task descriptions, so you can now see the complete field name on hover.
- **[UI]** Fixed archived processes not opening properly, so you can now access all your archived tasks in the new Tasks view filter.
- **[UI]** Fixed magic link launches that were incorrectly failing due to hidden formatting affecting process name character limits.
- **[UI]** Fixed rendering issue where rich text formatting and emoticons in field descriptions now display properly instead of appearing blank.
- **[UI]** Fixed AI step suggestions in procedure templates so you get fresh, helpful ideas when building your workflows instead of seeing no suggestions or the same ones repeatedly.
- **[UI]** Form field validations now work correctly for each individual field instead of accidentally copying rules between different fields.
- **[UI]** Fixed guest email validation in assignee fields so you can now assign tasks to people outside your organization without needing to include internal team members.
- **[UI]** Form field updates now display instantly for guests without needing to refresh the page.
- **[UI]** Fixed automation dropdown menus to prevent selecting the same step twice, making "if this then that" rules clearer and less confusing to set up.
- **[UI]** Fixed a technical error that occasionally prevented the help support widget from loading properly for some users.
- **[UI]** AI-generated step descriptions in templates now display completely without cutting off mid-sentence, giving you fuller guidance for each step.
- **[UI]** Fixed public form inviter selection to show "Select user" instead of code when the previously chosen user is deactivated.
- **[UI]** Fixed file uploads so deleted files now disappear immediately without needing to refresh the page when reopening tasks.
- **[UI]** Fixed Tasks view to automatically show all sections expanded, so you can see your tasks immediately without clicking to open each section.
- **[UI]** Fixed misleading "Saving changes" indicator that appeared when navigating document templates with form fields, even when no actual changes were made.
- **[UI]** Fixed starred items showing incorrectly - favorites now sync properly between different pages so you'll see the right status everywhere.
- **[UI]** Fixed the tracker view getting stuck on an endless loading screen when opened from a process page.
- **[UI]** Fixed column labels and dropdown indicators in AI-generated templates to make milestones and steps easier to navigate and understand.
- **[UI]** Fixed task completion button now correctly appears disabled for unassigned users in duplicated tasks, preventing confusion about permissions.
- **[UI]** Fixed dropdown arrows to show proper pointer cursor and improved click behavior for easier navigation.
- **[API]** Fixed complete button incorrectly appearing enabled for users without permission to complete restricted tasks.
- **[API]** Duplicated sub-tasks now appear correctly instead of staying permanently hidden, ensuring you can see and work on all your tasks.
- **[API]** Template duplication now correctly preserves the order of milestones, ensuring your duplicated templates maintain the exact same sequence as the original.
- **[API]** You can now search for groups by name instead of scrolling through the entire list.
- **[API]** Public forms now automatically stay accessible when the person who originally shared them is deactivated, ensuring uninterrupted form submissions.
- **[API]** Fixed an issue where some previously favorited templates and processes weren't appearing in your Favorites menu after our recent upgrade.

### Deprecated

- **[UI]** Explored advanced sunburst visualization for template libraries but prioritized simpler navigation improvements like better filters and view options instead.

### Security

- **[UI]** Fixed public form sharing to automatically switch ownership when an inviter is deactivated, keeping your forms accessible while maintaining security.

### Performance

- **[UI]** Fixed slow loading and freezing issues when viewing large tables and downloading files, making the app much more responsive.
