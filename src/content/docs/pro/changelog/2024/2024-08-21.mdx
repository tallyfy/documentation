### Bug Fixes

- **[UI]** Fixed template launching when guest assignment steps were incorrectly blocking launch instead of defaulting to the person launching the process.
- **[UI]** Fixed tooltip display for form fields so helpful text now shows completely instead of being cut off at window edges.
- **[UI]** Organization switching page now displays properly on mobile devices, with better text wrapping for long organization names and descriptions.
- **[UI]** Added "Remove" button for group logos so you can easily revert back to default initials after uploading a custom image.
- **[UI]** Fixed comment box visibility and functionality issues on Android tablets, ensuring seamless task collaboration across all devices.
- **[UI]** Fixed task completion to wait for file uploads to finish, preventing incomplete tasks and ensuring all required fields are properly filled before marking tasks as done.
