### Changes

- **[UI]** Major UI improvements and API enhancements are coming soon after comprehensive quality testing to ensure the best experience.
- **[UI]** Made task panes open and close more smoothly on slower devices by optimizing animations for better performance.

### Bug Fixes

- **[UI]** Email steps can now be marked as milestones, giving you better visibility into important communications within your workflows.
- **[UI]** Fixed profile menu not appearing on mobile devices - now you can easily access your account options without the sidebar unexpectedly closing.
- **[UI]** Fixed an issue where converting archived guest users to Light members didn't properly add them to your member list.
- **[UI]** Fixed guest user interface issues including removing unsupported options and eliminating excessive scrolling in process views.
- **[UI]** Fixed Dutch translation errors to improve the experience for users in the Netherlands.
- **[UI]** Fixed the "Show Hidden Tasks" button so it now properly displays your hidden tasks in the Action Plan view when clicked.
- **[UI]** Fixed Active Process view to automatically load all tasks when opening deep-linked tasks, eliminating confusion from missing earlier tasks.

### Removed

- **[UI]** Improved scrollbar visibility across the app makes it easier to spot when content can be scrolled in any view.
