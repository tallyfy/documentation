### Changes

- **[UI]** Improved checkout page layout with better spacing and cleaner text display for easier payment processing.
- **[UI]** Made creating new roles in step assignments easier with a clearer dropdown menu that instantly shows your new role without refreshing the page.
- **[UI]** Trial users now see a helpful prompt to create their first template when clicking Library, making it easier to get started building workflows.

### Bug Fixes

- **[UI]** Guest users now only see tasks assigned to them, improving privacy and reducing clutter in their task view.
- **[UI]** Fixed excessive network requests when navigating between pages, making the app faster and more responsive.
- **[UI]** Fixed favorites feature to instantly highlight templates and processes without page refreshes or visual glitches.
- **[UI]** Fixed task completion to wait for file uploads to finish, preventing form corruption and ensuring required fields stay properly validated.
- **[UI]** Fixed duplicate text showing twice when setting up automation rules in procedure templates, making the interface cleaner and easier to read.
- **[UI]** Fixed broken "View" links for public templates in Examples folder - you can now properly browse example templates again.
- **[UI]** Fixed missing dropdown arrows in step settings and corrected arrow direction when collapsing webhook sections for clearer navigation.
