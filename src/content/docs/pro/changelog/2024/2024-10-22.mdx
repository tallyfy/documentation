### New Features

- **[UI]** Automation rules now support Assignee Picker fields, letting you automatically assign tasks based on who was selected in previous steps.

### Changes

- **[UI]** Template creation is now faster - you can grab multiple step ideas in a row without the ideas section closing after each selection.
- **[UI]** You can now choose who appears as the sender when sharing public forms, instead of it always showing the template creator's name.
- **[UI]** You can now see who has edit access to any template and request permission with one click, making collaboration much smoother.

### Bug Fixes

- **[UI]** Fixed the Support button in the sidebar so you can now submit questions and get help directly within the app again.
- **[UI]** Fixed file upload errors for users with Gmail plus addressing (like user+tag@gmail.com) in the process document upload section.
- **[UI]** Fixed print button to instantly open print dialog when clicked, restoring the smooth one-click printing experience for templates and processes.
- **[UI]** Fixed task reassignment after deactivating users - you can now properly assign active tasks to new team members without errors.

### Deprecated

- **[UI]** Fixed a selection issue in Tracker presets while preparing to launch our new bookmarks feature that will make process filtering even easier.

### Security

- **[UI]** Enhanced email security by requiring password verification when changing your email address, preventing unauthorized account access.
