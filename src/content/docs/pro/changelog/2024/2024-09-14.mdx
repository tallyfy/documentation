### New Features

- **[UI]** You can now customize which procedure template new team members receive during onboarding, or disable automatic onboarding entirely.
- **[UI]** Administrators can now disable AI features organization-wide to meet compliance requirements, giving enterprises full control over AI usage.

### Changes

- **[UI]** Made privilege role selection clearer by adding helpful descriptions for Administrator, Standard, and Light roles directly in dropdown menus.
- **[UI]** Email subject lines for guest task assignments now use the task title by default, ensuring clear and consistent communication without confusion.
- **[UI]** Added email suggestions and user icons when inviting guests during template launch to make it easier to find and select the right people.
- **[UI]** Streamlined the coupon code section during checkout with clearer layout and helpful text explaining discount eligibility.
- **[UI]** Simplified the pricing checkout page by removing duplicate headings, making it cleaner and easier to complete your purchase.
- **[UI]** Added helpful descriptions directly in the notification frequency dropdown so you can instantly understand what Electric, Mindful, and Chilled settings do without leaving the page.
- **[UI]** Streamlined the user reactivation process by improving existing communication options, making it easier for disabled users to request account access.
- **[UI]** Improved our integration strategy to use smarter middleware connections instead of direct third-party tools for better flexibility.

### Bug Fixes

- **[UI]** Fixed blank guest task links in the Settings > Guests section so you can now properly view and share links with external collaborators.
- **[UI]** Step assignment now displays suggested usernames clearly without overlapping text, making it easier to assign tasks to team members.
- **[UI]** Fixed timing dropdown options that disappeared after a recent update, ensuring all automation settings remain fully visible and accessible.
- **[UI]** Fixed infinite loading screen that prevented guest users from viewing their favorited items and navigating the app properly.
- **[UI]** Fixed favorites list for guests so items are properly removed when unfavorited instead of reappearing after page refresh.
- **[UI]** Fixed field count indicator to update instantly when adding or removing fields instead of requiring a page refresh.
- **[UI]** Fixed radio button visibility issue in step checklist validation settings so you can now clearly see which option is selected.
- **[UI]** Fixed creator name now showing properly next to "Added by" label in one-off tasks for better task attribution.
- **[UI]** Simplified comment box behavior in procedure templates to show consistent placeholder text, eliminating confusion when clicking different areas.
- **[UI]** Template permissions now default to the creator only, giving you better control over who can access your newly created templates.
- **[UI]** Fixed the "Create New Milestone" popup being cut off on mobile devices - it now appears centered and fully visible.
- **[UI]** Fixed processes with apostrophes in task names now loading properly in Tracker view instead of showing infinite loading.
- **[UI]** Fixed process names with apostrophes now displaying correctly in Tracker's Table view after appearing blank.
- **[UI]** Fixed missing lock icons and properly disabled action buttons for restricted tasks, so non-assigned users now clearly see when they can't take action.
- **[UI]** Fixed visual alignment of pointer arrows in popup windows when editing job titles and milestones for a cleaner, more polished interface.
- **[UI]** Fixed step edit menu dropdown positioning in mobile Tracker view so it appears in the correct location within step modals.
- **[UI]** Fixed deadline displays to show consistent, accurate time formats across all task views instead of confusing approximations.
- **[UI]** Improved multi-select dropdowns in automation rules to display selected items clearly without overlapping on all screen sizes.
- **[UI]** Light role users can now see available template options when clicking "Create Template" to help them understand upgrade benefits.
- **[UI]** Fixed missing visual shadows in the Automation tab across all template views for a more polished interface experience.
