### New Features

- **[UI]** Added an optional file name field for URL uploads in workflows, letting you give descriptive names to linked documents instead of showing long web addresses.
- **[UI]** You can now reset guest links with a dedicated Reset button without disabling the guest, making link management much simpler.

### Changes

- **[UI]** Form fields in invite modals now show consistent coloring that prioritizes clear validation feedback over branding colors.
- **[UI]** Guest users now see pending tasks first by default with clearer filter controls, making it easier to focus on what needs attention.

### Bug Fixes

- **[UI]** Fixed public process sharing to properly display all tasks and content instead of just the process name when using the "PUBLIC TO THE WORLD?" setting.
- **[UI]** Fixed task statuses in Tasks view now updating instantly to show correct completion status without needing to refresh the page.
- **[UI]** Fixed button layout in Power Tools so they stay properly positioned when validation messages appear during user assignments.
- **[UI]** Fixed tasks view freezing with endless loading after updates, plus improved scrolling in template editor for smoother navigation.
- **[UI]** Fixed page scrolling in template editor that would get stuck after canceling launch wizard on longer templates.
- **[UI]** Fixed self-assign button showing incorrectly for Standard users on restricted tasks, preventing confusing permission errors when trying to assign themselves.
- **[UI]** Fixed email task displays to show clearer completion status and replaced confusing "UNACKNOWLEDGE" button with "RE-OPEN" for better clarity.
