## How to insert media into a procedure's description

Tallyfy allows the following media to be inserted into any procedure's description:

- Images
- Files(100MB limit)
- Videos
- Links
- Tables
- URLs

To insert any of them, simply go to the **Edit view** of your desired template, and select the task you want to insert media to. Then, simply click on the **Insert** option, and select which media you want to embed!
