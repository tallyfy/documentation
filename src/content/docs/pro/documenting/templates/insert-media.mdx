## Adding media to procedure descriptions

Tallyfy allows you to enhance task descriptions with various media types, providing more detailed instructions and visual aids.

## Supported media types

You can insert the following media into any procedure description:

* Images
* Files (100MB maximum size)
* Videos
* Links
* Tables
* URLs

## Insertion process

To add media to a task:

1. Open the template in **Edit** view
2. Select the task you want to enhance
3. Click the **Insert** button
4. Choose the media type from the dropdown menu
5. Upload or link the media according to the selected type
