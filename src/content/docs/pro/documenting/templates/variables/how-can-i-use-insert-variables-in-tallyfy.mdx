## Everything about variables

Welcome to Tallyfy's help article on using Insert Variables! ðŸŒŸ

Insert Variables are a powerful way to ensure that every user sees the right information at the right time in their tasks. With real-time updates, you can keep task descriptions up-to-date, even when information changes. Let's dive in and learn how to make the most of Insert Variables.

### Setting Up Insert Variables in Step Titles

To get started, follow these simple steps:

1. Open your desired template in Edit mode.
   ![Image](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-open-edit-mode-template.png)
2. Create the necessary form fields in an independent task that will be used in a later dependent step title.
   ![Image](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-show-formfields.png)
3. In the dependent step title, click on the `{ }` button to insert the variable.
4. Launch your process, and voila! The step title will dynamically display the form value in its description.

### Setting Up Insert Variables in Step Descriptions

Now, let's see how to set up Insert Variables in step descriptions:

1. Open your desired template in Edit mode.
2. Create the necessary form fields in an independent task that will be used in a later dependent step.
3. In the dependent step, create a description and insert the form variable using the `{ }` button in the right-hand corner.
   ![Image](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-insert-variables.png)
4. Launch your process, and just like magic, the dependent task will display the form value in its description.
