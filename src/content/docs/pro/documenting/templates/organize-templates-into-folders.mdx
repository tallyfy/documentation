## How to organize templates into folders

Folders help you efficiently organize templates for easier navigation and management. With proper organization, you can quickly find templates and access favorites.

## Create folders for your templates

To create a folder:
1. Navigate to the Templates section
2. Click the **New Folder** button
3. Enter a name for your folder

![Create Folder](https://screenshots.tallyfy.com/tallyfy/pro/desktop-light-create-folder-template.png)

After creation, the folder appears in the left panel. Click on it to view contained templates.

![Point folder](https://screenshots.tallyfy.com/tallyfy/pro/desktop-light-point-folders-templates.png)

## Edit folders

To edit a folder:
1. Select the folder
2. Click the pencil icon next to the folder name
![Edit Folder](https://screenshots.tallyfy.com/tallyfy/pro/desktop-light-click-edit-folder-template.png)

You can:
* Rename the folder
* Change its parent folder
* Delete the folder (templates will be preserved)

![Options edit folder](https://screenshots.tallyfy.com/tallyfy/pro/desktop-light-edit-folder-templates.png)

## Move templates to folders

To move a template to a folder:
1. In the Library view, click the â‹® icon on the template
2. Select **Move**
3. Choose an existing folder or create a new one

![Move to Folder](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-move-template-to-folder1701482910854.png)

When you delete a folder, templates within it are preserved and returned to the main library.

## Unarchive templates

When you unarchive a template, it returns to its original folder location.

![Unarchive Template](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-show-archived-templates1701483092924.png)

## Create subfolders

For more granular organization, create subfolders within existing folders:
1. Select a parent folder
2. Click **New Folder**
3. Name your subfolder

![Subfolders](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-create-subfolders.png)
