## How to print, download, and export Tallyfy templates

This article explains how to access, print, download, and export templates in Tallyfy.

## Viewing templates

To view a template:

1. Select **Templates** from the left navigation panel
2. Click the **View** icon next to the desired template
   ![View template](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-click-view-template.png)
3. Scroll through the template content

You can switch between viewing mode and editing mode using the toggle at the top of the page.

## Printing templates

To print a template:

1. Open the template in Template Editor
2. Click **More > Print**
   ![Print option](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-click-print.png)
3. Select your printer in the **Destination** field
4. Follow your browser's printing instructions

## Downloading templates

To download a template as a PDF:

1. Open the template in Template Editor
2. Click **More > Print**
3. In the print dialog, set **Destination** to "Save as PDF" or "Save to Google Drive"
4. Complete the save process according to your browser's instructions

## Exporting templates via API

For advanced users, Tallyfy provides API capabilities for template export:

1. Access our [GitHub repository](https://github.com/tallyfy/api-support/tree/master/Backup%20Blueprints)
2. Use the provided Python script to import and export templates programmatically
3. Follow the documentation in the repository for implementation details
