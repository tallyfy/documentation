## Organizing with tags

Tags provide a flexible categorization system that helps organize, filter, and locate templates and processes in Tallyfy.

### Prerequisites
- Administrator access (to manage organization-wide tags)
- Template or process editing permissions (to apply tags)

### Benefits of using tags

- **Improved searchability** - Find specific templates and processes quickly
- **Flexible organization** - Categorize items using multiple criteria simultaneously
- **Visual identification** - Color-coded tags provide visual cues
- **Dynamic filtering** - Combine tag filters to narrow down specific items
- **Process management** - Group related processes regardless of folder structure

### Tag management

Tags can be centrally managed through organization settings:

1. Navigate to **Settings > Organization > Tags**
2. To create a new tag, click **Add Tag**
3. To edit an existing tag:
   - Click on the tag name to modify its text
   - Click on the color swatch to change its color
   - Click **Save** when finished
   ![Editing a tag](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-edit-tag.png)
4. To delete a tag:
   - Click the ellipsis (â‹®) icon next to the tag
   - Select **Delete Tag**
   - Confirm the deletion

### Tagging templates

Add tags to templates for better organization and discoverability:

1. Open the template in **Edit** mode
2. Click **Settings** in the toolbar
3. Select **Summary** from the dropdown
4. In the **Tag** field:
   - Type to search existing tags
   - Select from suggested tags
   - Create a new tag by typing a unique name
5. Click **Save Changes**

![Adding a tag to a template](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-add-tag-to-template.png)

### Tagging processes

Apply tags to active processes for categorization and filtering:

1. Open the desired process
2. Click the **Add Tag** button near the process name
3. Select from existing tags or create a new one
4. Click outside the tag selector to apply

![Adding a tag to a process](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-add-tag-process.png)

### Filtering by tags

Locate specific items using tag filters:

1. Navigate to the **Tracker** section
2. Click **Add filter**
   ![Accessing the filter menu](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-click-filter.png)
3. Select **Tags** from the filter criteria
4. Choose one or more tags from the dropdown menu
   ![Filtering processes by tag](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-filter-process-by-tag.png)
5. The list will update to show only items with the selected tags

### Best practices for tagging

- **Consistent naming** - Establish a naming convention for tags
- **Limited number** - Use 10-15 primary tags to avoid overcomplication
- **Logical grouping** - Create tags that represent meaningful categories:
  - Department (HR, Finance, Marketing)
  - Process type (Approval, Review, Onboarding)
  - Priority (High, Medium, Low)
  - Status (Active, Testing, Archived)
- **Combine with folders** - Use tags to complement, not replace, folder organization
- **Regular maintenance** - Periodically review and clean up unused tags
