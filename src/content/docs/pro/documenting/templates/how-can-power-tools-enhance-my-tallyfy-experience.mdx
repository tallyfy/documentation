## Using Powertools for template management

Tallyfy's Powertools feature provides productivity enhancements for template creation and management. These tools streamline common tasks through batch operations.

Access Powertools via the **Config** section in the Template Editor.

## Available Powertools

### Bulk step creation

Create multiple steps at once:
1. Enter step titles by copy-pasting or typing each on a new line
2. Click **BUILD** to generate all steps simultaneously

![Bulk step creation](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-click-powertools-create-steps.png)

### Batch assignment

Assign multiple steps to team members in a single operation:
1. Select the team member from the dropdown menu
2. Click **Assign** to create the assignment
3. Click **APPLY** when all assignments are complete

![Batch assignment](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-click-powertools-assign.png)

### Bulk deadline adjustment

Modify multiple deadlines simultaneously:
1. Specify the time adjustment (add or subtract minutes, hours, days, weeks, or months)
2. Click **UPDATE** to apply the changes to all selected deadlines

![Bulk deadline adjustment](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-click-powertools-deadlines.png)

### Template favoriting

Add templates to team members' favorites lists:
1. Enter the names of team members in the text box
2. The template will appear in the favorites tab for those users

![Template favoriting](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-click-powertools-favorite.png)
