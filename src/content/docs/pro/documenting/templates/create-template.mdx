## How to create templates in Tallyfy

Templates are the building blocks of your processes in Tallyfy. They are the reusable, customizable, and scalable workflows that you can use to automate your business processes.

No matter how you choose to create a template, you'll need to follow these steps:

1. Click on the **Create** button at the top of the left side bar.
2. Select **Create a template** from the dropdown menu.
3. Choose one of the three ways to create a template.

![image](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-click-create-fullscreen.png)

There are three ways to create a template in Tallyfy:

1. From scratch
2. Using Tallyfy AI
3. Uploading a file to Tallyfy AI

## Create a template from scratch

Creating a template from scratch means that you will start with a blank canvas and build your process from the ground up. This is a great option if you have a unique process that you want to document and automate. To start creating a template from scratch, simply select the **From scratch** option and give your new template a name. You will then be redirected to the template editor where you can start building your process the way you want it.

![image](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-create-template-scratch.png)

## Create a template using Tallyfy AI

Sometimes you might not know where to start when creating a template. That's where Tallyfy AI comes in. Tallyfy AI is a powerful tool that can help you create a template. To create a template using Tallyfy AI, simply select the **Tallyfy AI** option. Select the type of template you're looking to create (Procedure or documents). You will then need to give your template a meaningfull name, aswell as the type of company you have and the language you want the template to be in. so that our AI can understand what you're looking to create. Once you've done that, click on the **Create** button and Tallyfy AI will generate a template for you. Simply review the template and make any changes you need to before saving it.

![image](https://screenshot.launchbrightly.com/tallyfy/pro/desktop-light-create-client-onboarding-template-from-tallyfyAI.png)

## Create a template by uploading an SOP document

If you already have an SOP document that you want to turn into a template, you can do that by selecting the **From SOP document** option. You will then need to upload the SOP document you want to turn into a template. Your PDF must not be password protected or encrypted, and the maximum size of the document should be 30Mb.

Tallyfy will generate a template based on the content of the SOP document. You can then review the template and make any changes you need to before saving it.

![image](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-create-from-sop.png)
