## How to see all activity in a template

The Activity stream feature provides a comprehensive log of all actions performed on templates. This tracking capability helps monitor changes and maintain accountability.

## What activities are tracked

The activity log captures a wide range of template-related actions:
* Updates to template titles and descriptions
* Changes to kickoff form fields
* Archiving and unarchiving templates
* Tag additions and modifications
* Process instruction creation and edits
* Webhook management
* Other template configuration changes

## Viewing the activity log

To access the activity trail for a template:

1. Navigate to the **Templates** view
2. Select the template you want to monitor
3. Click **View > Activity**

![Activity Tab](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-check-template-activity.png)

4. The chronological activity log displays all recorded actions

## Access permissions

Only Administrators and members with explicit access permission to the template can view its Activity Trail.
