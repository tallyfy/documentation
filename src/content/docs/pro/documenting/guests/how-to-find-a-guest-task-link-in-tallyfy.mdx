## Finding and sharing guest task links

[Guest](/products/pro/documenting/guests/) task links provide external users access to specific [tasks](/products/pro/tracking-and-tasks/tasks/) without requiring a Tallyfy account. These links remain valid even after the guest completes the task.

### Prerequisites
- A [task](/products/pro/tracking-and-tasks/tasks/) assigned to a [guest](/products/pro/documenting/guests/) user
- Administrator or Standard [member](/products/pro/documenting/members/) permissions

### Method 1: From the task menu

1. Navigate to the [process](/products/pro/tracking-and-tasks/processes/) containing the guest task
2. Locate the task assigned to the guest
3. Click the ellipsis (â‹®) icon next to the task
4. Select **Copy Guest Link** from the dropdown menu

![Guest link in task menu](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-copy-guest-link.png)

### Method 2: From the Organization Settings

For a centralized view of all guest links:

1. Navigate to **Settings > Organization > [Guests](/products/pro/documenting/guests/)**
2. Find the guest user in the list
3. Click **Copy Guest Link** in the guest's row

![Guest link in settings](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-copy-guest-link-from-settings.png)

### Sharing the link

After copying the link:
- Paste it into an email, message, or other communication tool
- The [guest](/products/pro/documenting/guests/) can use this link to access their assigned task without logging in
- The link remains valid throughout the task's lifecycle
