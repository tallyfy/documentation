## The guest dashboard

The guest dashboard provides external users with a centralized view of all tasks assigned to them across multiple processes.

### Accessing the guest dashboard

Guests can access their dashboard in two ways:
1. Through the **unique task link** sent via email
2. By visiting [https://go.tallyfy.com](https://go.tallyfy.com) and entering their email address

### Dashboard features

The guest dashboard includes:

- **Process list** - All processes containing tasks assigned to the guest
- **Task cards** - Individual tasks requiring the guest's attention
- **Status indicators** - Visual cues showing the status of each task (pending, in progress, completed)
- **Comment section** - Area to provide feedback or ask questions about tasks
- **Attachment viewer** - Access to files shared as part of tasks
- **Due date indicators** - Time remaining to complete tasks

### Navigating the dashboard

#### Viewing process details
1. Locate the process card in the dashboard
2. Click on the process name to expand details
3. View all tasks within that process assigned to you

#### Working with tasks
1. Select the task you want to view or complete
2. Review the task description and requirements
3. Complete any form fields or uploads requested
4. Add comments if needed
5. Click **Complete** when finished

#### Tracking progress
- Completed tasks display a checkmark
- Current tasks are highlighted
- Upcoming tasks show when they will become active
- Process completion percentage indicates overall progress

### Guest permissions

As a guest user, you can:
- View and complete your assigned tasks
- Access task-related documents
- Add comments on your tasks
- Track progress of processes where you're involved
- Receive email notifications about tasks
