## Creating a table of contents

Tallyfy automatically generates a table of contents based on heading structure, providing easy navigation through documents and procedure templates.

### Prerequisites
- A document or procedure template in edit mode
- Content to organize with headings

### Creating a table of contents in documents

The table of contents is created automatically as you add headings to your document:

1. Select the text you want to designate as a main section
2. Click the formatting dropdown in the toolbar
3. Select **H1** to create a top-level heading
   ![Creating an H1 heading](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-mark-heading-h1-example.png)

4. For subsections, select the appropriate text
5. Choose **H2** or **H3** from the formatting dropdown
   ![Creating an H2 heading](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-mark-h2-heading.png)

The table of contents will automatically populate with your headings in the navigation panel.

### Heading hierarchy best practices

For optimal table of contents organization:
- Use **H1** for main sections only
- Use **H2** for subsections within H1 sections
- Use **H3** for further subdivision when needed
- Maintain a logical hierarchy without skipping levels
- Keep heading text concise and descriptive

### Table of contents in procedure templates

Procedure templates also contain a table of contents for easier navigation:

1. Open a procedure template
2. Click **Preview** to view the template as it will appear to users
3. Navigate to the table of contents by clicking the TOC icon in the upper right
   ![Accessing procedure table of contents](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-where-to-click-table-contents-procedure-template.png)
4. Use the table of contents to jump to specific sections
   ![Procedure template table of contents](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-table-of-contents-procedure-template.png)

### Benefits of using a table of contents
- Provides a clear visual overview of document structure
- Enables quick navigation to specific sections
- Encourages logical organization of content
- Makes lengthy documents more accessible
- Helps users understand the document flow at a glance
