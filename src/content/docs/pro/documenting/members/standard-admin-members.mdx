## Member role comparison

Tallyfy has two primary member roles with different permission levels and capabilities: Administrator and Standard.

### Administrator members

Administrator members have complete control over the Tallyfy platform and can perform all actions available in the system.

#### Administrator capabilities
- **Organization management:**
  - Invite new members
  - Edit billing information
  - Update plan subscriptions
  - View/modify organization details
  - Access all templates and processes
  
- **Member management:**
  - Set permissions for all members
  - Edit member profiles
  - Remove members
  - View activity logs
  
- **Template/process control:**
  - Create, edit, and view all templates
  - Set template and process access permissions
  - Delete archived templates
  - Manage all aspects of process execution

#### Finding administrators
To view all administrator members:
1. Navigate to **Settings > Organization > Members**
2. Look for members with the **Administrator** role

### Standard members

Standard members have limited permissions focused on day-to-day operational tasks rather than administrative functions.

#### Standard member capabilities
- **Template management:**
  - Create, edit, and view permitted templates
  - Organize templates into folders
  - Import templates (if permitted)
  
- **Organization interaction:**
  - Invite other standard members (if permitted)
  - View organization details like members list and guests
  - Access tags, custom branding, and snippets
  - View tasks and email logs of other members
  
- **System access:**
  - View API Access Token
  - Update custom branding settings and logo

#### Standard member restrictions
Standard members cannot:
- Set template and process access permissions
- Delete archived templates
- Access organization and member profiles
- View plan and billing information
- Modify subscription details
