## How to delete an archived process

To delete a [process](/products/pro/tracking-and-tasks/processes/) in Tallyfy, follow these simple steps:

<Steps>
1. Open the **[Tracker](/products/pro/tracking-and-tasks/tracker-view/)** view from the sidebar.
2. Add a new filter to show **Archived** processes.
   ![Image](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-filter-archived-processes.png)

3. Find the archived process you want to delete and click on **Delete**.
</Steps>

Remember, only archived [processes](/products/pro/tracking-and-tasks/processes/) can be deleted. For help with archiving a process, see [this article](/products/pro/tracking-and-tasks/tracker-view/organize-processes/how-to-archive-and-unarchive-processes-in-tallyfy/).
