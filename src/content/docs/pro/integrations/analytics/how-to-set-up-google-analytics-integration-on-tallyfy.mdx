## Engagement Tracking with Analytics Platforms

Tallyfy integrates with popular analytics platforms to help you measure how users engage with documentation, processes, and templates. This guide explains how to set up Google Analytics and Mixpanel integrations to capture detailed engagement metrics.

### Integration Benefits

- **User Behavior Analysis** - Track how users interact with process templates
- **Engagement Measurement** - Quantify time spent on specific documentation
- **Adoption Metrics** - Identify most and least utilized processes
- **Usage Patterns** - Discover when and how teams access information
- **ROI Measurement** - Correlate documentation usage with business outcomes

## Google Analytics Setup

### Prerequisites

- Google Analytics account with admin access
- Tallyfy Professional or Enterprise plan
- Administrator permissions in Tallyfy

### Step 1: Configure Custom Dimensions

1. Log in to your Google Analytics account
2. Navigate to **Admin** > **Property** > **Custom Definitions** > **Custom Dimensions**
3. Create five custom dimensions with the following configuration:

| Name | Index | Scope | Active |
|------|-------|-------|--------|
| User Name | 1 | User | Yes |
| User Email | 2 | User | Yes |
| Process Name | 3 | Session | Yes |
| Event Type | 4 | Hit | Yes |
| Event Details | 5 | Hit | Yes |

![Custom Dimensions Configuration](https://screenshots.tallyfy.com/file-z6yfQpYc2p.png)

:::caution[Exact Configuration Required]
Ensure you enter the custom dimension names, indices, and scopes exactly as shown. Incorrect configuration will result in missing or corrupted data.
:::

### Step 2: Obtain Tracking ID

1. In Google Analytics, navigate to **Admin** > **Property Settings**
2. Copy your Tracking ID (format: UA-#####-#)

### Step 3: Configure Tallyfy Integration

1. In Tallyfy, navigate to **Settings** > **Integrations**
2. Select the **Analytics** section
3. Enter your Google Analytics Tracking ID in the designated field
4. Click **Save**

![Tallyfy Google Analytics Integration](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-google-analytics-integration.png)

After completing these steps, Tallyfy will automatically send event data to your Google Analytics account.

## Enhanced User Tracking with User ID

To track specific users across sessions and devices, you can enable the User ID feature in Google Analytics.

### Step 1: Enable User ID View

1. In Google Analytics, navigate to **Admin**
2. In the **Property** column, select **Tracking Info** > **User ID**

![Admin navigation](https://screenshots.tallyfy.com/file-22ItrMHbDh.png)

3. Toggle the User ID feature to **ON**
4. Follow the terms of service prompts

![User ID section](https://screenshots.tallyfy.com/file-rKqFJRFFom.png)

### Step 2: Create User ID View

1. Still in the User ID section, follow the setup wizard (steps 1-3)
2. Create a new reporting view specifically for User ID data
3. Click **Create**

![User ID setup](https://screenshots.tallyfy.com/file-jnmcCbtx8O.png)

:::note[Data Processing Delay]
According to Google, there may be a 24-48 hour delay before data appears in your new User ID view.
:::

## Mixpanel Integration

For more granular event tracking, Tallyfy also supports integration with Mixpanel.

### Mixpanel Configuration

1. In Tallyfy, navigate to **Settings** > **Integrations** > **Analytics**
2. Enter your Mixpanel Project Token
3. Click **Save**

### Mixpanel Data Insights

Mixpanel provides detailed event categorization for specific actions within Tallyfy:

- Template views
- Step completions
- Process launches
- Documentation engagement
- User-specific activities

![Mixpanel Event Categories](https://screenshots.tallyfy.com/file-lfm4qkBjDC.png)

:::caution[Privacy Considerations]
When using Mixpanel integration, you are responsible for managing privacy compliance. Mixpanel collects all event data in your Mixpanel instance, which may include personally identifiable information. Ensure your privacy policies account for this data collection.
:::

### Interpreting Analytics Data

Once your integration is active, you can:

1. Create custom reports in Google Analytics or Mixpanel
2. Set up dashboards to monitor engagement trends
3. Configure alerts for significant changes in usage patterns
4. Export data for further analysis in other business intelligence tools

For specialized analytics requirements or custom implementation support, contact Tallyfy support.
