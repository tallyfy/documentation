## What is Tallyfy Analytics and how does it work?

Tallyfy Analytics represents an advanced add-on service that securely replicates your comprehensive workflow process data to dedicated analytical infrastructure for sophisticated business intelligence analysis and strategic decision-making.

This powerful service enables seamless integration with leading BI tools including Tableau, Power BI, and Looker, facilitating advanced data analysis and custom reporting capabilities that extend far beyond standard Tallyfy interface limitations.

### What key features does Tallyfy Analytics provide?

- **Comprehensive Process Data Access**: Retrieve detailed, granular information from all your organizational workflows and business processes.
- **Advanced SQL Query Capabilities**: Execute complex analytical questions about your data using industry-standard SQL syntax and functions.
- **Business Intelligence Tool Integration**: Seamlessly connect with enterprise reporting tools that support Amazon Athena connectivity.
- **Custom Dashboard Creation**: Design sophisticated reports and interactive charts tailored to your organization's specific analytical needs.
- **Trend Analysis and Performance Tracking**: Monitor and analyze how business processes perform and evolve over time with historical data insights.

### How does Tallyfy Analytics function operationally?

- **Premium Add-on Service**: Available as an advanced add-on service, typically for customers on annual subscription plans.
- **Investment Required**: Additional cost structure applies (contact Tallyfy for current pricing and implementation details).
- **Secure Private Infrastructure**: Your organizational data is stored securely in your own dedicated private environment using Amazon Athena (part of AWS cloud infrastructure).

:::note[Data Collection Starts After Activation]
Tallyfy only starts collecting and processing data for Analytics *after* you have purchased and activated the add-on. It does not include data from before activation.
:::

### What security credentials does Tallyfy Analytics provide?

When Tallyfy Analytics is configured for your organization, Tallyfy provides two types of secure authentication credentials:

1. **AWS Console Administrative Login**: Enables administrators to access the AWS web console for advanced configuration and management purposes.
2. **API Access Key & Secret Key Pair**: These credentials are used specifically to establish secure connections between your BI tools (such as Power BI or Tableau) and your Tallyfy data stored in Amazon Athena.

### How can I get started with Tallyfy Analytics?

To begin your Tallyfy Analytics implementation, please contact Tallyfy support or your dedicated account manager for configuration assistance and pricing information.
