## Tallyfy Analytics Overview

Tallyfy Analytics is an add-on service that connects your process data with business intelligence (BI) platforms, enabling comprehensive data analysis and custom reporting.

### Key features:

- **Real-time process data** - Access complete workflow information
- **SQL-based querying** - Run complex queries against your process data
- **BI platform compatibility** - Works with any tool supporting Amazon Athena drivers
- **Custom dashboard creation** - Build organization-specific visualizations
- **Historical trend analysis** - Track performance over time

### Implementation details:

- Available as an add-on for annual plans
- Priced at $20 per active member per year
- Data stored securely in your dedicated Amazon Athena instance
- Connects with popular BI tools including Tableau, PowerBI, and Looker

:::note[Data Collection Activation]
Data processing and collection for Tallyfy Analytics begin **only after** the add-on has been activated for your organization. Historical data prior to activation is not included.
:::

### Credential types

When Tallyfy Analytics is provisioned for your organization, you'll receive two sets of credentials:

1. **AWS IAM login credentials**
   - Used exclusively for web access to the AWS console
   - Enables administrative configuration of your data warehouse

2. **Access Key and Secret Access Key**
   - Required for connecting BI tools to Amazon Athena
   - Used in ODBC/JDBC connection configuration

Contact Tallyfy support to enable Analytics for your organization.
