### Required Headers

Ensure your request includes the necessary headers for authentication and identification:

- `Authorization: Bearer YOUR_ACCESS_TOKEN` (See [Authentication Methods](../../how-to-integrate-with-tallyfy-using-api/#authentication-methods))
- `Content-Type: application/json`
- `X-Tallyfy-Client: APIClient`
