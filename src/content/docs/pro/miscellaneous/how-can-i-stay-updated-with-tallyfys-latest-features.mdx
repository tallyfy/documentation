## How to find Tallyfy<sup>速</sup> latest updates

There are two main ways to see what's new in Tallyfy<sup>速</sup>:

1.  **Inside Tallyfy<sup>速</sup>**: Click your name in the top-right corner, then choose **Latest Updates** from the menu.
2.  **On the Website**: Go directly to the Tallyfy<sup>速</sup> changelog page: [tallyfy.com/changelog](https://tallyfy.com/changelog/)
