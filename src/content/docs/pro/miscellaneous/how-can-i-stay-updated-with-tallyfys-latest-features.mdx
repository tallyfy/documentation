## How can I stay updated with Tallyfy<sup>®</sup> latest features?

Tallyfy<sup>®</sup> regularly releases new features and improvements. You can stay informed about these updates through two convenient methods:

### Where can I view the latest updates within Tallyfy?

**Inside the Tallyfy<sup>®</sup> Platform**: Click your name in the top-right corner, then select **Latest Updates** from the dropdown menu to see recent platform improvements and new features.

### Where can I access the complete changelog online?

**On the Tallyfy Website**: Navigate directly to the comprehensive changelog page at [tallyfy.com/changelog](https://tallyfy.com/changelog/) to view detailed release notes and feature announcements.
