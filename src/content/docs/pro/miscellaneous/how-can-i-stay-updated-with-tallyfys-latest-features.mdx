## Check out our latest features

There are two ways to access the latest updates and new features in Tallyfy:

1. **Via the Tallyfy Platform**: Click on your name in the top right corner, and then select "Latest Updates".

2. **Through our Website**: Visit our changelog page at [https://tallyfy.com/changelog/](https://tallyfy.com/changelog/) to see all the features recently added.
