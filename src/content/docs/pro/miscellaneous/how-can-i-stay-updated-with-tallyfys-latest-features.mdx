## How to find Tallyfy's latest updates

There are two main ways to see what's new in Tallyfy:

1.  **Inside Tallyfy**: Click your name in the top-right corner, then choose **Latest Updates** from the menu.
2.  **On the Website**: Go directly to the Tallyfy changelog page: [tallyfy.com/changelog](https://tallyfy.com/changelog/)
