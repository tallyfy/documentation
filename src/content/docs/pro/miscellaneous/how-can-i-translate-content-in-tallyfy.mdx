## How can I translate content in Tallyfy<sup>¬Æ</sup>?

Tallyfy<sup>¬Æ</sup> enables multilingual teams to collaborate effectively through two comprehensive translation approaches:

1.  **Interface Language Customization**: Modify the display language for Tallyfy<sup>¬Æ</sup> menus, buttons, and system interface text
2.  **Automatic Content Translation**: Translate user-generated text (including task descriptions and comments) using Microsoft Azure AI technology (requires administrator configuration)

### How can I change my interface language?

This feature modifies the language displayed for Tallyfy's standard interface elements, affecting only your personal view of the platform.

<Steps>
1. Click the globe icon (üåê) located in the top menu bar.
2. Select your preferred language from the available language list.
3. Tallyfy's interface will immediately switch to display in that language.
</Steps>

### What interface languages are available?

Tallyfy supports numerous interface languages including English, Spanish, French, German, Chinese, Japanese, and many additional options. Contact support if your preferred language isn't currently available.

### How can administrators set up automatic content translation?

This powerful feature enables real-time translation of user-written text, facilitating seamless collaboration across language barriers. The system utilizes Microsoft Azure Cognitive Services and requires administrator configuration for activation.

#### What languages does automatic translation support?
Automatic content translation supports over 100 languages through Microsoft Azure's comprehensive translation services.

#### What requirements are needed for automatic translation setup?

*   Active Microsoft Azure account with translation services enabled
*   API Key and Region/Location credentials for the Azure Translator service
*   Administrator-level access within your Tallyfy organization

#### How do I configure automatic translation in Tallyfy?
*(For comprehensive Azure configuration steps, refer to the [Azure setup guide](/products/pro/integrations/azure-translation/how-to-set-up-azure-cognitive-ai-integration/).)*

<Steps>
1.  **Azure Configuration**: Create a "Translator" service resource and obtain the **API Key** and **Region/Location** credentials.
2.  **Tallyfy Configuration**: Navigate to **Settings** > **Integrations** > **Translation** within your organization settings.
3.  **API Key Entry**: Paste your Azure **API Key** into the designated API Key field.
4.  **Region Configuration**: Enter your Azure service **region** identifier.
5.  **Optional Setup**: Enter your Azure Subscription Key if applicable to your configuration.
6.  **Activation**: Click **Save** to complete the translation service setup.
</Steps>

### How do users access translation after setup?

Once administrators successfully establish the Azure connection, the globe icon in the top menu bar enables all users to select their preferred language for automatic content translation.

### How can I use automatic content translation?

After administrators successfully configure the Azure translation connection, all users can access this feature:

<Steps>
1. Click the globe icon (üåê) in the top menu bar.
2. Select the target language you want content translated into.
3. Green globe icon indicates Azure Cognitive Services is connected and actively translating content.
4. View tasks, comments, and other content as Tallyfy automatically translates text written by others into your selected language.
</Steps>

:::note[Translation limitations]
Real-time translation works best with standard business text. Highly technical terms, slang, or specific industry jargon might not translate accurately.
:::
