## Where can I find the in-app user guide?

Tallyfy<sup>Â®</sup> includes a comprehensive built-in help system that provides instant access to answers and step-by-step guides while you work. This integrated support ensures you can resolve questions without leaving your current workflow.

Two primary access methods are available when logged into Tallyfy:

<Steps>
1.  **Sidebar Help Button**: Click the **Help** button (displaying a **?** icon) positioned near the bottom of the left sidebar.
    *   This action opens a comprehensive support panel. Use the `Answers` tab to search through help articles, or select the `Ask` tab to send direct messages to our support team.
        ![Help button in sidebar and the resulting panel](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-access-user-guide.png)

2.  **User Profile Menu**: Click your name or profile picture in the top right corner, then select **Help and FAQs** from the dropdown options.
    *   This option opens the complete help documentation website in a new browser tab for comprehensive reference.
        ![Help and FAQs option in the user menu](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-access-help-and-faqs.png)
</Steps>
