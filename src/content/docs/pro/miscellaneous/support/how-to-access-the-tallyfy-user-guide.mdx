## Finding the in-app user guide

Tallyfy has a built-in user guide, so you don't need to leave the app to find answers and step-by-step guides. There are two ways to access it:

<Steps>
1.  **Sidebar Help button** - Click the **?** icon near the bottom of the left sidebar.
    *   You'll see a support panel. Use the `Answers` tab to search help articles, or the `Ask` tab to message our support team directly.
        ![Tallyfy sidebar help button and support panel](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-access-user-guide.png)

2.  **Profile menu** - Click your name or profile picture in the top right corner, then select **Help and FAQs**.
    *   This opens the full help documentation site in a new browser tab.
        ![Tallyfy profile menu with Help and FAQs option](https://screenshots.tallyfy.com/tallyfy%2Fpro%2Fdesktop-light-access-help-and-faqs.png)
</Steps>
