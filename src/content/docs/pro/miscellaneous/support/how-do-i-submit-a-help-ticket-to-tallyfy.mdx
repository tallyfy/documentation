## How to submit a Tallyfy help ticket

If you need help from Tallyfy Support with a question, problem, or feature idea, there are two main ways to submit a ticket:

1.  **Email**: Send an email directly to `support@tallyfy.com`.
2.  **Help Bubble**: Click the green help bubble icon located in the bottom right corner of most Tallyfy pages. This lets you search help docs or send a message to create a support ticket.
