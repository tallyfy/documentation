## How to use keyboard shortcuts in Tallyfy

This article lists keyboard shortcuts and tips to help you navigate Tallyfy.

### Keyboard shortcuts

Keyboard shortcuts allow you to perform actions faster. Here are some shortcuts you can use in Tallyfy:

#### +Create button

You can use these shortcuts anywhere in the app (except from within a [process](/products/pro/tracking-and-tasks/processes/) or [template](/products/pro/documenting/templates/)).

- To quickly add a one-off [task](/products/pro/tracking-and-tasks/tasks/), press **CTRL + OPT + T** simultaneously.
- To enter the template library and [launch a process](/products/pro/launching/), press **CTRL + OPT + P** simultaneously.

#### Completing form fields

When completing a [task](/products/pro/tracking-and-tasks/tasks/) form, you can use the following shortcuts:

- Press **TAB** to navigate from _[form field](/products/pro/tracking-and-tasks/tasks/what-are-form-fields-in-tallyfy/)_ to _form field_. You can also use it to travel within form field options.
- Press **SPACEBAR** to select a given option:
  - Dropdown
  - Radio Button
  - Checklist
  - File Upload
