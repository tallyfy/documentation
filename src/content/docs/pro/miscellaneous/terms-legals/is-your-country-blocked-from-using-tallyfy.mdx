## Restricted countries for Tallyfy access

Tallyfy is a US corporation. To comply with US law, access from certain countries is currently blocked. We apologize for any inconvenience.

The locations currently restricted from accessing Tallyfy are:

*   `CU` - Cuba
*   `IR` - Iran
*   `SY` - Syria
*   `KP` - North Korea
*   `LY` - Libya
*   `SO` - Somalia
*   `IQ` - Iraq
*   `RU` - Russia
*   `YE` - Yemen
*   `PS` - Palestine
*   `T1` - Tor exit nodes (anonymous network, not a country)

If you have questions about potential exceptions or need assistance, please [contact us](https://tallyfy.com/contact-us/).
