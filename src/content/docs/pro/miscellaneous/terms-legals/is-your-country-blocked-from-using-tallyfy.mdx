## Restricted countries for Tallyfy access

Tallyfy is a US corporation. To comply with US law, certain countries are currently blocked from using Tallyfy. We apologize for any inconvenience this may cause. The country codes below are [here](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2#Officially_assigned_code_elements).

The countries currently restricted from accessing Tallyfy are:

* 'CU' - Cuba
* 'IR' - Iran
* 'SY' - Syria
* 'KP' - North Korea
* 'LY' - Libya
* 'SO' - Somalia
* 'IQ' - Iraq
* 'RU' - Russia
* 'YE' - Yemen
* 'PS' - Palestine
* 'T1' - Tor exit nodes (not a country)

If you have questions about exceptions or if you need assistance, please [contact us](https://tallyfy.com/contact-us/).
