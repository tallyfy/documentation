## How to clear your cache in Chrome(Chromium)

Clearing the cache in Chrome can help resolve some of the most common issues with web applications such as Tallyfy. Here's how you can do it:

1. Open Chrome.
2. Click on the three dots in the top right corner of the browser window.
3. Select **Clear browsing data**.
4. In the dialog box that appears, select the time range you want to clear the cache for. You can choose from the last hour, the last 24 hours, the last 7 days, the last 4 weeks, or **All time**, aswell as the types of data you want to clear. **We recommend checking all the boxes**.
5. Click on **Clear data**.

:::note[Getting more information]

If you want to learn more about clearing the cache in Chrome, you can visit the [official Google support page](https://support.google.com/accounts/answer/32050?co=GENIE.Platform%3DDesktop).

:::

That's it! Your cache is now cleared, and you should be able to use Tallyfy without any issues. If you're still experiencing problems, please feel free to [contact us](https://tallyfy.com/contact-us/). We'll be more than happy to assist you!
