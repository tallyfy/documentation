## How to clear your cache in Edge

Clearing the cache in Edge can help resolve some of the most common issues with web applications such as Tallyfy. Here's how you can do it:

1. Open Edge.
2. Click on the three dots in the top right corner of the browser window.
3. Select **Settings**.
4. Scroll down and click on **Privacy, search, and services**.
5. Under **Clear browsing data**, click on **Choose what to clear**.
6. Select the types of data you want to clear. **We recommend checking all the boxes**.
7. Click on **Clear now**.

:::note[Getting more information]

If you want to learn more about clearing the cache in Edge, you can visit the [official Microsoft support page](https://support.microsoft.com/en-us/microsoft-edge/clear-cookies-and-site-data-in-microsoft-edge-63947406-40ac-c3b8-57b9-2a946a29ae09).

:::

That's it! Your cache is now cleared, and you should be able to use Tallyfy without any issues. If you're still experiencing problems, please feel free to [contact us](https://tallyfy.com/contact-us/). We'll be more than happy to assist you!
