## Getting started with Tallyfy

Welcome to Tallyfy! This section provides simple, step-by-step guides to help you begin using the platform.

### Recommended tutorials for new users

We suggest new users follow these guides in order:

1.  [Create an organization](/products/pro/tutorials/how-can-i-create-a-new-tallyfy-account/) - Set up your Tallyfy account and company workspace.
2.  [Navigate Tallyfy](/products/pro/tutorials/navigate-tallyfy/) - Get familiar with the main screens and menus.
3.  [Create your first template](/products/pro/tutorials/your-first-template/) - Learn how to build your first process blueprint.
4.  [Launch your first process](/products/pro/tutorials/your-first-process/) - Start running a process from your template.

### What you'll learn

These tutorials cover the basics:

*   Setting up your account.
*   Designing workflows (templates).
*   Managing tasks.
*   Working with your team.
*   Using simple automations.
