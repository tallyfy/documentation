## Getting Started with Tallyfy

Welcome to Tallyfy! This section provides step-by-step tutorials to help you quickly become productive with the platform.

### Tutorial path for new users

For best results, follow these tutorials in sequence:

1. [Create a new Tallyfy account](/products/pro/tutorials/how-can-i-create-a-new-tallyfy-account/) - Set up your organization
2. [Navigate Tallyfy](/products/pro/tutorials/navigate-tallyfy/) - Learn the interface and basic navigation
3. [Create your first template](/products/pro/tutorials/your-first-template/) - Design your first workflow template
4. [Launch your first process](/products/pro/tutorials/your-first-process/) - Put your template into action

### What you'll learn

These tutorials cover essential skills for Tallyfy success:

- **Account setup**: Create and configure your organization
- **Process creation**: Design efficient, repeatable workflows
- **Task management**: Assign, track, and complete tasks
- **Collaboration**: Share processes with your team
- **Automation**: Streamline repetitive work with rules and triggers
