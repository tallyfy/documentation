## How to create a new organization on Tallyfy

Creating a user account and creating an organization are separate steps in Tallyfy:

1. Visit [https://tallyfy.com/start/](https://tallyfy.com/start/)
2. Fill out the required fields and click submit
3. Using Microsoft or Google accounts is recommended for simplified authentication
4. Complete the registration process to create your account
