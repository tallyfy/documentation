## What are the Tallyfy Denizen API endpoints?

### Base URL

```
https://denizen.tallyfy.com/
```

### How do you use Tallyfy Denizen?

To use Tallyfy Denizen in your application:

1. Make a GET request to the base URL
2. Tallyfy Denizen automatically spots user location via IP address
3. An image that fits the detected location is returned

### What types of responses does Tallyfy Denizen provide?

When calling the Tallyfy Denizen API, you'll get one of two responses:

* **Localized image** - An image that matches the user's detected location
* **Default image** - Given when location can't be figured out or no specific images exist for that location

## What geographic coverage does Tallyfy Denizen provide?

### How does image availability change by region?

* **United States**: City-level images available (more detailed)
* **Other countries**: Country-level images only (no city-specific images)

### How is Tallyfy Denizen implemented technically?

Tallyfy Denizen is made for performance and reliability:

* Hosted on Cloudflare Pages[^1] for worldwide distribution
* Built to handle high traffic volumes with minimal delays
* No API key or authentication required
