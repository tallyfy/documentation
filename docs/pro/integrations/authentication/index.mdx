## Integrate with your Sign Sign On (SSO) system

Authentication middleware is a convenient way of granting access to multiple services with just one set of credentials.

Tallyfy offers seamless and _free_ integration with the most popular SAML and SSO platforms available. It's free because we believe basic security should not cost extra.
