## Integrate PowerBI with Tallyfy

Microsoft PowerBI is powerful data visualization software. Tallyfy Analytics feeds content from your process data in real-time to Amazon Athena. By connecting PowerBI to your Athena instance (provided by Tallyfy) you can build any visualization you choose.
