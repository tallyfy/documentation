## How to set up Tallyfy with Power Automate

Microsoft Power Automate and Tallyfy can complement each other in running effective workflows.

For official Microsoft Power Automate documentation related to Tallyfy, you can visit [this link](https://docs.microsoft.com/en-us/connectors/tallyfy/).

We made this a video to demonstrate how to use Power Automate to easily integrate with Tallyfy:

<lite-vimeo videoid="347324914"></lite-vimeo>

Getting started with Tallyfy and Power Automate is simple. All you need is your Tallyfy credentials to sign in.

Once you've created a one-time connection, you'll be able to access your Tallyfy account in all of your flows.
